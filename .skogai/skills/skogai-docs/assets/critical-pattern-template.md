______________________________________________________________________

## N. [Pattern Name] (ALWAYS REQUIRED)

### WRONG ([Will cause X error])


### CORRECT

```[language]
[code showing correct approach]
```

**Why:** [Technical explanation of why this is required]

**Placement/Context:** [When this applies]

**Documented in:** `docs/solutions/[category]/[filename].md`

______________________________________________________________________

**Instructions:**

1. Replace N with the next pattern number
1. Replace [Pattern Name] with descriptive title
1. Fill in WRONG example with code that causes the problem
1. Fill in CORRECT example with the solution
1. Explain the technical reason in "Why"
1. Clarify when this pattern applies in "Placement/Context"
1. Link to the full troubleshooting doc where this was originally solved
