🧠 Cleaner tool calls in Agent Chat


 If you live in Agent Chat, this one should make your runs a lot less noisy.

We redesigned the tool call UI so you can understand what’s happening without 
scrolling through a wall of logs:


 * Tool calls are now shown in compact, grouped rows with stacked icons
 * File edits and creations link straight into a diff panel for fast review
 * The whole timeline is more skimmable when you’re debugging complex agent 
workflows

Open Agent Chat, kick off a run, and you’ll see the new layout in action.

Learn more here. 
<https://docs.replit.com/updates/2026/01/16/changelog#redesigned-tool-calls-ui-in-agent-chat>


📌 Other new features


 * Node.js security patch: All Agent dev environments and hosted Node apps 
have been upgraded to secure Node versions, so you stay protected by 
default—just republish if it’s been a while.
 * Automation step graph preview: See your automation workflow graph in 
seconds instead of minutes, so you can catch wiring issues early and iterate 
faster on agents.
 * Private apps on free plans: Keep projects private even on the free tier, 
with a small “Made with Replit” badge that you can remove by upgrading toReplit 
Core.
 * replit ai in Shell + new models: Call Replit AI directly from Shell with the
replit ai command, and experiment with the latest GPT, Claude, and Gemini 
models through AI Integrations.
