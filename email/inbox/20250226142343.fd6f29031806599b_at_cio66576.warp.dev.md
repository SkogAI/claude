***************************
Introducing Warp on Windows
***************************

If you've been waiting to use Warp on your PC, the time has come. Our agentic, modern terminal experience now has full support for Windows developers ( https://www.warp.dev/windows-terminal ).

Get started today:

( https://app.warp.dev/get_warp )

***********************************
By installing Warp, you will unlock
***********************************

*
Built in AI ( https://www.warp.dev/ai-tools ): type or speak in natural language to Agent Mode in your terminal to debug errors, fix code, summarize lengthy logs, and more
*
Collaboration and sharing ( https://www.warp.dev/warp-drive ): Developers can share terminal sessions, save commands in their personal or team drives, and share their workflows via link with Warp users (and non-users)
*
Modern UX: ( https://www.warp.dev/modern-terminal ) grouped input and output blocks and IDE-like text editing allow you to more easily navigate your command line. We have support for Powershell, WSL, and Git Bash
* Bonus AI: 2x AI requests for your first month, exclusively for new Warp on Windows users
