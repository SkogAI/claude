Product updates


Set a budget limit & alerts in the dashboard

You can now set a limit in the dashboard [https://e2b.dev/dashboard] and if you hit it, new API requests get blocked. You'll get
email notifications at 50%, 80%, 90%, and 100% of your limit. You can also set up custom budget alerts. Sign in at E2B
[https://e2b.dev/dashboard] and try it.
 

https://e2b.dev/dashboard

 


Environment variable inside the sandbox

There is now an environment variable [https://e2b.dev/docs/sandbox/environment-variables] (E2B_SANDBOX=true) set inside every E2B
Sandbox.
If you have a custom template, you must rebuild it for this feature to work.
 


You can now watch nested directories.

You can enable recursive watching [https://e2b.dev/docs/filesystem/watch#recursive-watching] using the parameter recursive.

https://e2b.dev/docs/filesystem/watch#recursive-watching

https://e2b.dev/docs/filesystem/watch#recursive-watching

Built with E2B


Smolagents by Hugging Face

Hugging Face allowed web browsing agents to their smolagents library [https://huggingface.co/docs/smolagents/index]. It unlocks
building use cases like agentic web browsing, and it offers secure code execution powered by E2B.
 


BigCodeBench

This open evaluation platform [https://bigcode-bench.github.io/] on AI for software engineering allows real-time code execution
for any program, powered by E2B sandboxes.
 


Open computer use

Secure AI computer use powered by E2B Desktop Sandbox. Supports 10+ LLMs, OS-Atlas [https://osatlas.github.io/]/ShowUI,
[https://github.com/showlab/ShowUI] and any other models you want to integrate
[https://github.com/e2b-dev/open-computer-use#llm-support].
 

https://github.com/e2b-dev/open-computer-use
