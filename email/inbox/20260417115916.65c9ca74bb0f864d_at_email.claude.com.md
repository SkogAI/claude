 Introducing Claude Opus 4.7 


 Claude Opus 4.7 holds context better over long sessions, recovers from errors 
more reliably, and remembers more across sessions so you can prompt less and 
get more.


 With high-resolution image support and stronger design sense, Opus 4.7 
generates higher quality UI from a description or reference image.


 Opus 4.7 is the new default model in Claude Code. We’ve also updated the 
default effort level toxhigh, the best setting for complex coding tasks. For 
simpler tasks, and for balancing intelligence and cost, we recommend switching 
tohigh effort (/effort). See more best practices 
<https://claude.com/blog/best-practices-for-using-claude-opus-4-7-with-claude-code?utm_content=inline_link>
 for using Opus 4.7 with Claude Code.



 Learn more  
<https://www.anthropic.com/news/claude-opus-4-7?utm_content=hero_cta> 

 New ways to build with Claude Code 



 <https://claude.com/blog/auto-mode?utm_content=image_link> 


 Run longer tasks while Claude handles permissions 


 Instead of approving every file write and bash command or skipping 
permissions entirely, auto mode lets Claude handle permission decisions for you 
using classifiers to check for certain risky actions.


 Give Claude Code a large task with upfront context, then let it run in auto 
mode.


 Available now in research preview for Team, Enterprise, API, and Max users. 
Select auto mode with Shift+Tab.



 Learn more  <https://claude.com/blog/auto-mode?utm_content=secondary_cta> 


 <https://claude.com/blog/claude-code-desktop-redesign?utm_content=image_link> 


 A redesigned desktop app for parallel work 


 We’ve redesigned Claude Code on desktop 
<https://claude.com/blog/claude-code-desktop-redesign?utm_content=inline_link> 
so that you can run multiple agents and review their work in a single window.


 You now get an integrated terminal, in-app file editing, HTML/PDF previews, 
and a faster diff viewer—all in a drag-and-drop layout that you can arrange the 
way you want.


 Download the Claude desktop app to get started. 



 Get the app  <https://claude.com/download?utm_content=hero_cta> 




 Routines that run automatically 


Routines 
<https://claude.com/blog/introducing-routines-in-claude-code?utm_content=inline_link>
 (research preview) let you configure prompts, repos, and connectors once—then 
trigger them on a schedule, via API, or on a GitHub webhook.


 They run on Claude Code’s web infrastructure 
<https://code.claude.com/docs/en/claude-code-on-the-web?utm_content=inline_link>
, so nothing depends on your laptop being open.


Create a routine <https://claude.ai/code/routines?utm_content=secondary_cta> 





 Review without breaking focus 


/ultrareview spins up a review session that reads changes and flags what a 
careful reviewer might catch, directly in your terminal. You get three free 
ultrareviews to try it out, with additional reviews available pay-as-you-go via
extra usage 
<https://support.claude.com/en/articles/12429409-manage-extra-usage-for-paid-claude-plans?utm_content=inline_link>
.


 Try it in your terminal 





 Hand Claude a task from your phone 


 Use Dispatch 
<https://code.claude.com/docs/en/desktop?utm_content=inline_link#sessions-from-dispatch>
 (research preview) to kick off a build, run tests, or put up a PR from your 
phone. Claude runs it in your local machine with the desktop app. You’ll get 
pinged when it’s done or needs your go-ahead.


 Available on Pro and Max plans.
