Launch Log 1 is here
********************

Q1 has been one of the busiest periods in Warp’s history. Check out the release video where Jess and Ian share the newest features.

( https://www.youtube.com/watch?v=sjbaWZfgbyg )Read on for a summary of what's new:

*
Agent Mode Prompts: You can now save natural language Agent Prompts in your Warp Drive. Is there a question or task you keep asking Agent Mode to help you with, or a Prompt you would love to share with your peers? You can now save it to avoid re-typing every time. We've compiled a list of our favorites at dothings.warp.dev ( https://dothings.warp.dev/ ). Take a look, and add your own!
* Knowledge: You can now make rules so Warp can remember what tools you have installed and how your dev environment is set up. Have tools you prefer using, style preferences in your code, or even want to make Warp Agents more or less verbose? Make a rule.
*
Codebase as context: Warp AI is getting smarter— we will now share metadata about your current repo to make responses better. Metadata will include things like languages, directory structure, and filenames. Note, no code is ever stored on Warp's servers.
* The long (but important) tail: Choose your app icon on MacOS, memory improvements, and more!

More details can be found on our blog ( https://www.warp.dev/blog/launch-log-1 ). Thanks for using Warp, and let us know what you think on X ( https://x.com/warpdotdev ), where you can join our new community, or by filing a GitHub ( https://github.com/warpdotdev/Warp ) issue.

Happy coding,

The Warp Team
