Hey Skogix

We just wrapped up our 13th Launch Week and have a lot to share with you so let’s get right to it:

## Day 1 - AI Assistant v2
Now available in the Supabase Dashboard, the upgraded Assistant offers powerful tools like  schema design, data querying, debugging, RLS policy creation, and more. Just hit cmd+i or click the Assistant icon to try it out! 
[[Blog post](https://supabase.link/ai-assistant-v2-email)]
[[Video](https://supabase.link/ai-assistant-v2-video)] 

Day 2 - **Edge Functions Upgrade**
We’ve made huge improvements to Supabase Edge Functions. You can now handle long-running processes, use temporary file storage for operations like zip file extraction, and establish real-time WebSocket connections.
[[Blog post](https://supabase.link/edge-functions-upgrade)]
[[Docs]](https://supabase.link/edge-functions-docs)
[[Video](https://supabase.link/edge-functions-video)]

## Day 3 - Supabase Cron
With easy scheduling via the Dashboard or SQL, you can run jobs like database maintenance, generating reports, triggering webhooks, or executing custom logic with Edge Functions. 
[[Blog post](https://supabase.com/blog/supabase-cron)]
[[Docs](https://supabase.com/docs/guides/cron)]
[[Video](https://supabase.link/cron-post)]

## Day 4 - Queues
Supabase Queues is a Postgres-native, durable Message Queue with guaranteed delivery, improving the scalability and resiliency of your applications. Queues let you manage tasks like sending emails, processing images, or generating embeddings asynchronously. 
[[Blog post](https://supabase.link/queues-post)]
[[Docs](https://supabase.link/queues-docs)]
[[Video](https://supabase.link/queues-video)]

## Day 5- database.build v2: Bring-your-own-LLM
We’ve upgraded **database.build** with the ability to Bring your own LLM. Connect any OpenAI-compatible model for more flexibility and control. And you can now deploy what you create on database.build to Supabase in one click.
[[Blog post](https://supabase.link/byo-llm)]
[[Video](https://supabase.link/byo-llm-video)]
