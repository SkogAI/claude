Store API keys or passwords in encrypted vaults and reuse them securely in all your policies.

```
ngrok api vaults create --name "my-vault"
ngrok api secrets create --name "password" --value "hunter22" \
--vault-id="vault_1234"
```

```
on_http_request:
  - actions:
      - type: basic-auth
        config:
          credentials:
            - "alan:${secrets.get('my-vault', 'password')}"
```
