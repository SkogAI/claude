Hey there,
 
Roo Code Cloud has been evolving fast, so we wanted to tell you what's new in the past couple of weeks.
 
New Features
 * Cloud Agents are now available for everyone, without the need for a subscription. If you haven't yet, go set up your Cloud
   Agent team [https://app.roocode.com/cloud-agents/setup] (it's free to try and takes seconds).

 * The Cloud Agent team has grown! The Explainer, Planner and Coder agents have joined the PR Reviewer and the PR Fixer agents.
   You can now build entire features, from idea to merged PR, without ever opening your IDE.

 * We made it easier to set up your Cloud Agents right from the sidebar. Clicking on them lets you see their tasks, quickly change
   the model they use and start new tasks.

 * The Roo Code Cloud Provider is now available to everyone! It's a curated set of models which work great with Roo, with no need
   to mess with API keys. Includes free models and paid ones at a low cost.

 * Added Gemini 3 Pro, Claude Opus 4.5 and Haiku 4.5 to the Roo Code Cloud Provider and added a page to browse available models
   [https://app.roocode.com/models].

 
Improvements
 * We revamped the Slack integration so now it's faster and smoother to get tasks going from there. It's awesome to see threads
   turn into PRs and questions from non-technical folks get answered right there.

 * We improved web push notifications so you can get updates on your tasks right in your browser (on desktop or mobile device).

 * For tasks started on the web, we now send emails reporting completion status, so nothing gets lost.

 * Cleaned up all of the New Agent Tasks screens and added some ideas, so it's easier for folks to try them out.

 * "Thinking" blocks in messages are now collapsible to keep your chat clean, and we remember your preference.

 * You can now see exactly who on your team created a task in the Task Details section.

 * Added Z.Ai as a provider option.

 
Bug Fixes
 * Fixed issues where the task view wouldn't auto-scroll or show the "Scroll to Bottom" button and some scrolling problems on
   mobile.

 * Fixed an issue where interactive task notifications weren't being sent.

 * Cleared a permission issue preventing cloud jobs from accessing paid models despite active billing.

 * Improved reliability of credit auto-top ups [https://app.roocode.com/billing] when balances get low.

 
Head on over to your Roo Code Cloud account [https://app.roocode.com] to check these out. We'd love to know what you think! Reply
to give us your thoughts (real humans read every message).
 
Happy building,
 
- Bruno and the Roo Code Team
 
PS: We updated our website [https://roocode.com] to try to explain to more people why Roo is awesome. What do you think?
