рҹ—әпёҸ Roadmap 2025

В 
WeвҖҷve charted the course [https://langfuse.com/roadmap] for Langfuse this year:
В 
 * Agent Observability: Improved support for agentic applications with advanced visualizations, filtering for tool calls, and
   standardized agent evaluations.

 * Tracing: Introduce environment separation, real-time LLM data streaming, and OpenTelemetry support for broader trace
   instrumentation.

 * Prompt Management: Diff view, custom RBAC, LLM-assisted prompt engineering, prompt composability, placeholders, and native tool
   call support to streamline prompt workflows.

 * Evaluation: non-LLM evaluators, repeated span analysis, metric comparisons, and session-level scoring.

 * Query 100s of Millions of Traces: Large-scale trace management with full-text search and self-serve dashboards for custom
   metrics.

 * Data Platform: Webhooks, batched exports, and rule-based routing to integrate and automate data flows.

 * Other: Enhance UI/UX, simplify onboarding, and introduce project-level data retention policies for a smoother overall
   experience.

В 
We are continuously updating our roadmap page [https://langfuse.com/docs/roadmap] with more information.В
