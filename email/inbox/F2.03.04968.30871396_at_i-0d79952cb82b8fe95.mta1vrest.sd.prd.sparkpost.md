Let's jump in 👇




🧑‍🎨 Design Mode



Design Mode <https://docs.replit.com/replitai/design-mode#design-mode>, 
powered byGemini 3, lets you create beautiful interactive designs and static 
websites faster than you can order a coffee.

 Design Mode is perfect for rapid prototyping, landing pages, or just getting 
your ideas out of your head and onto a screen. And when you're ready to add 
real functionality?One click converts your design into a full app.

No more choosing between "looks good" and "works well." Now you get both!


⚡ Fast Mode



 Speed shouldn't be a premium feature, that's why we added Fast Mode 
<https://docs.replit.com/replitai/fast-mode#fast-mode>.

Fast mode is a lightning-quick version of Agent optimized for targeted 
changes. Get visual tweaks, bug fixes, and reasonable-sized features in 10-60 
seconds instead of waiting minutes. Perfect for rapid iteration and pair 
programming.

 Try it out and see what you've been missing.


💰 Stripe Integrated Payments



Want to charge money for your app? Now it's easier than ever.

Our one-click Stripe integration handles:


 *  Automatic payment and subscription setup
 * Real-time sync to your Replit Database 
<https://docs.replit.com/cloud-services/storage-and-databases/sql-database#database>
 for subscription statuses 

No wrestling with webhooks. No manual status checks. Just connect Stripe 
<https://docs.replit.com/replitai/stripe-payments#stripe-payments-integration> 
and start building a business.

📌 Other new features


 * Agent tools dropdown in Plan Mode: Configure autonomy level and preferences 
during the initial planning step
 * `@` filename support in Agent: Mention files with @ alongside / commands
 * Import zip archives directly: Upload .zip files that automatically extract 
into new Replit Apps
 * Replit Auth UI improvements: Redesigned authentication flow with a cleaner, 
more polished interface
 * Enhanced Atlassian connectors: Jira and Confluence now auto-detect site 
URLs for easier setup
 * HoundDog privacy scanning: Detects PII, API keys, and credential exposure 
in your codebase
 * Todoist connector: Manage Todoist tasks programmatically from your Replit 
apps
 * Better SEO for deployments: Improvements to custom domains improve SEO for 
Replit Apps
