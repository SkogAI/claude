Hi Skogix,

We are sharing a few updates to Letta's pricing.


NEW: MAX PLANS

Based on community feedback, we're introducing Max plans https://www.letta.com/pricing ($200/month) to support power users of Letta Code. The Max plan include usage across all model providers with higher rate limits, and can also be combined with direct API pricing for flexible pay-as-you-go overage.


SIMPLER USAGE-BASED PRICING

Due to confusion about the current step-based pricing for agents, we are moving to token-based pricing. When interacting with agents, you will be charged (in credits) for token usage at API prices (see details on the models page https://app.letta.com/projects/default-project/models). Letta Pro plans include at least $20 in model inference at API prices. Credit usage will also be show in dollar amounts.

BYOK usage stays the same: use /connect in Letta Code to easily connect your own API keys free of charge, which includes external subscriptions like ChatGPT Plus/Pro (Codex) and the GLM + MiniMax coding plans.
