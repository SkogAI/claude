🖼️ Multimodal Knowledge Base
============================

Your knowledge base can now handle images. Starting from [v1.11.0](https://b6usd.r.ag.d.sendibm3.com/mk/cl/f/sh/7nVU1aA2nfsTSeY62oltNKjy2JTL2Wc/r-ChNkf_A8_Q), diagrams, screenshots, and photos are embedded into the same semantic space as text. You can search by description or upload an image directly—the system retrieves matching visuals and text together.


When paired with a Vision-enabled LLM, agents can reference retrieved images in their reasoning. Useful for product manuals, technical documentation, or anything where visuals carry real information.

[Learn more →](https://b6usd.r.ag.d.sendibm3.com/mk/cl/f/sh/7nVU1aA2nfuMSB3P1O0LPh3mCfy7mT8/9Xqvkh_oLAy5)

**🤖 Agentic RAG**
=================

Traditional RAG is one-shot: retrieve once, then generate. Agentic RAG adds a reasoning loop: the agent analyzes intent, chooses tools, rewrites queries, evaluates evidence, and retries or falls back until the context is strong.

We published a guide on how this works in Dify using the Agent Node. Worth a read if you're building anything that needs reliable, multi-step retrieval.

[Read the guide →](https://b6usd.r.ag.d.sendibm3.com/mk/cl/f/sh/7nVU1aA2nfy8RE40yWTFUPhOXOxhGMA/dzaSW3owNiDa)
