The Runpod Hub makes it easy to deploy prebuilt AI environments — no setup required. It’s where developers and researchers can launch production-ready endpoints using GitHub-native repos.

One of the latest drops? Pruna AI’s FLUX.1-dev-juiced, a radically optimized version of the FLUX.1-dev diffusion model. This repo makes image generation faster, cheaper, and greener — with impressive compile-time optimizations across caching, quantization, and torch compilation.

***************
Why it matters:
***************

*
Generates images in as little as 3.76s using “Smashed” optimization mode
* Integrates caching, quantization, compilation, and factorization via torchao, fora, and more
* Ideal for developers who want high-performance inference with zero infra complexity
* Hosted on the Hub ( https://console.runpod.io/hub?createListing=false ) — deploy it straight from GitHub, no DevOps required

*******************
How to get started:
*******************

* Go to the FLUX.1-dev-juiced ( https://console.runpod.io/hub/PrunaAI/runpod-worker-FLUX.1-dev ) repo on Runpod Hub
* Click “Deploy runpod-worker-FLUX1-dev” in the upper right
* Choose your GPU and region
*
Start generating images via API — no additional config required

Midjourney has launched its first AI video generation model, transforming the image-focused platform into a multimedia powerhouse. V1 animates still images into 4-second videos with options to extend up to 16 seconds total. The model works with both Midjourney-generated and uploaded external images, offering automatic or manual animation modes.
