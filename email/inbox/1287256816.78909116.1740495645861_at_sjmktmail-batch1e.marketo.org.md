Materialized views is a commonly used feature within ClickHouse that allows you to query pre-computed data based on a SELECT statement. Materialized views speed up your queries by preventing calculations from having to be evaluated every time you run a query for common access patterns.

Creating a materialized view is easy. Simply use the CREATE MATERIALIZED VIEW command like so:

CREATE MATERIALIZED VIEW analytics.monthly_aggregated_data_mv
  TO analytics.monthly_aggregated_data
  AS
SELECT
    toDate(toStartOfMonth(event_time)) AS month,
    domain_name,
    sumState(count_views) AS sumCountViews
FROM analytics.hourly_data
GROUP BY
    domain_name,
    month

This query will store your precomputed query results into a target table called `monthly_aggregated_data`, from which you can query your data quickly without any expensive calculations.
