&#128680;  Release 1.41
Notable changes



	- &#128306; st.metric and st.columns have a parameter to show an optional border.

	- &#127912; Text and background color in Markdown can use "primary" color from the theme.primaryColor configuration option.

	- &#129398; You can freeze columns with column configuration when scrolling horiztontally.

	- &#128280; The type parameter for buttons accepts a new option, "tertiary".

	- &#128694; Support for pathlib.Path objects everywhere you can use a string path.



See the full changelog <https://go.snowflake.net/MjUyLVJGTy0yMjcAAAGXfu0GibKdY4MPg7P2PYays4hxVZFVjSjfXdmZc52uk2FEmAjxzBTgLdLRSUuQ1SGbU_I0xks=> for more details and download the latest version.
