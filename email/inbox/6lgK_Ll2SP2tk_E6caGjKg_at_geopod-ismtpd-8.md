-----------------------------------------

Agent mode in multi-file edits helps GitHub Copilot gather context across multiple files, suggest and test edits, and validate changes with speed and accuracy. Also check out Next Edit Suggestions and image support in Copilot Chat.

*To get started with Agent mode, download VS Code Insiders and then enable the agent mode setting for GitHub Copilot Chat.

DeepSeek-R1, OpenAI o3-mini, Gemini 2.0 Flash, and more are now available in GitHub Models
------------------------------------------------------------------------------------------

We’ve added multiple new models to GitHub Models, including DeepSeek-R1 ( https://github.com/marketplace/models/azureml-deepseek/DeepSeek-R1/playground?utm_campaign=02-11-2025-Product-Update-Email&utm_medium=email&utm_source=copy-link ) , OpenAI o3-mini ( https://github.com/marketplace/models/azure-openai/o3-mini/playground?utm_campaign=02-11-2025-Product-Update-Email&utm_medium=email&utm_source=copy-link ) , Phi-4 ( https://github.com/marketplace/models/azureml/Phi-4/playground?utm_campaign=02-11-2025-Product-Update-Email&utm_medium=email&utm_source=copy-link ) , and Codestral 25.01 ( https://github.com/marketplace/models/azureml-mistral/Codestral-2501/playground?utm_campaign=02-11-2025-Product-Update-Email&utm_medium=email&utm_source=copy-link ). You can try them out for free in the GitHub Models playground and compare their output side-by-side with other models, or experiment with them through the Models API. Plus: Gemini 2.0 Flash ( https://github.blog/changelog/2025-02-05-google-gemini-2-0-flash-is-now-available-to-all-copilot-users-in-public-preview/?utm_campaign=02-11-2025-Product-Update-Email&utm_medium=email&utm_source=copy-link ) is now available for use in GitHub Copilot.
