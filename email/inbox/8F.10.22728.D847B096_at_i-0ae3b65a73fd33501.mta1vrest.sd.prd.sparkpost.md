 * Connectors Platform with 25+ pre-built integrations—including Stripe,

& much more! Let's jump in 👇




🔌 Connectors are here



 We launched the Connectors Platform 
<https://docs.replit.com/replitai/integrations> this month with 25+ pre-built 
integrations that let Agent talk to the services you actually use.

 Connect to Stripe, Figma, ClickUp, Salesforce, Zendesk, and other services 
right from your workspace. Agent can now read your design files, create support 
tickets, update CRM records, or process payments—all without you leaving Replit.

 Discovery is simple: just type / in Agent or head to the integrations page 
from the side nav to see available connectors and integrate them instantly.

For Enterprise teams, we added role-based connector management so you can 
control shared API keys, configure custom OAuth clients, and manage which 
connectors are available at the org level.


🔨 Plan & Build Mode improvements



 We've made a number of improvements to how Agent works in Plan 
<https://docs.replit.com/replitai/plan-mode> and Build modes:


 * Clear design specs: Agent now generates detailed design guidelines so your 
app looks clean from the start
 * AI-generated images: Need a hero image or icon? Agent creates them inline
 * Stock image integration: Access stock photos without leaving your workspace
 * Production-ready code: Better component structure and cleaner output
 * Plan Mode is now more proactive. Agent suggests tasks before you ask!

Agent now also creates automatic checkpoints when you move from planning to 
building.

 Those checkpoints <https://docs.replit.com/replitai/agent#checkpoints> mean 
you can experiment freely and roll back changes if something doesn't work. It's 
like git commits, but Agent handles them for you at logical milestones.

 We also added an optional Build Score rating system (1-5) after key 
milestones like your first checkpoint or publishing. Your feedback helps us 
make Agent better!


🔒 Security and reliability upgrades



As always, we've made a number of security and reliability upgrades:


 * Dependency vulnerability scanning for Go, JavaScript, Python, and Rust
 * 7-day soft delete for databases with support-assisted restoration
 * Faster custom domain verification with clearer DNS guidance, automatic 
retry, and better troubleshooting
 * Redesigned domains interface that makes managing custom domains simpler

📌 Other new features



Builder experience:


 * Real-time Agent cost visibility in the chat header during active tasks
 * Unified webview logging so you can debug with the same logs Agent sees
 * Faster Agent stopping with reduced wait times when starting/stopping
 * Escape key shortcuts for exiting Visual Select mode or returning to Build 
Mode
 * 19 additional Google Fonts in App Themes including Crimson, EB Garamond, 
Figtree, and Source Sans Pro


Git improvements:


 * Better remote management for connecting and creating Git remotes
 * Easier switching between personal and organization workspaces
 * Improved code verification before deployment

Enterprise:


 * SSO support for Replit Auth: Enterprise employees can now use company SSO 
to log into Replit Auth-enabled apps
 * Support seat access: Add Replit Support as a non-billable Guest with scoped 
access to specific apps
 * Bulk app deletion for organizations with appropriate permissions and 
warnings for apps with active deployments

 You can find weekly updates and more in our changelog 
<https://docs.replit.com/updates>.
