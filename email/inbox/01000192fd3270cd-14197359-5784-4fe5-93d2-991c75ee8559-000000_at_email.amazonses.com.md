Support for interactive charts

We added support for interactive charts [https://e2b.dev/docs/code-interpreting/create-charts-visualizations/interactive-charts].
E2B now automatically detects charts when executing Python code withinrunCode() JavaScript or run_code() in Python. The Python
code must include Matplotlib charts. E2B sends the data of the chart back to the client, so you can then use your favorite chart
library and create custom interactive charts.
 

Get started [https://e2b.dev/docs/code-interpreting/create-charts-visualizations/interactive-charts]

Example: AI Analyst with interactive charts

Try interactive charts in the E2B Analyst [https://github.com/e2b-dev/ai-analyst] app. This free and open-source app allows you to
analyze your CSV files with Llama 3.1 and create interactive charts.




Desktop sandbox for secure computer use

We created the E2B Desktop Sandbox [https://github.com/e2b-dev/desktop] with a graphical environment that you can connect to any
LLM for the secure use of your computer.
 

Check out the example [https://github.com/e2b-dev/secure-computer-use]

 
The E2B Desktop Sandbox is an isolated cloud environment with a desktop-like interface powered by E2B [https://e2b.dev/]. While
LLM-powered computer use is still in its early days, it represents an exciting direction for AI agents.
