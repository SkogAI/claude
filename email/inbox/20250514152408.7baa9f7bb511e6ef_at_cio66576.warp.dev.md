*************************
AI experience and context
*************************

Attach URLs as prompt context. For example, reference specific API documentation or reference internal docs to setup config files

Restart past Agent Mode conversations— Warp will remember session context so you can jump back into a workflow without losing your train of thought

Receive notifications for Warp Agents. These trigger when agents want your approval or finish a new task
