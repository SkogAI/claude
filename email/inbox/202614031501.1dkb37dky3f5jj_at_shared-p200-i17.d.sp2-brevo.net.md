🧑‍💻 **Human Input Node**
========================

Not every workflow should run from start to finish without a person involved. The new Human Input node in [v1.13.0](https://b6usd.r.a.d.sendibm1.com/mk/cl/f/sh/7nVU1aA2ng01QkZJx5hhWm1ChlSU0Ig/jMJacLmLT_wu) lets a workflow pause, send a form to someone via Web App or Email, and continue down different branches depending on their response.


Use it to review AI-generated content before publishing, require approval on sensitive data, or hand off edge cases to a colleague. It works like any other node on the canvas: drop it in, connect it, and configure the form and decision buttons.


[Learn more →](https://b6usd.r.a.d.sendibm1.com/mk/cl/f/sh/7nVU1aA2ng1uQH4cvew9Z8L0s7xGkFC/AebskXBr0rbg)

📚 **Summary Index**
===================

Chunk-based retrieval works well for specific questions, but struggles with broader ones that span multiple chunks. Summary Index in [v1.12.0](https://b6usd.r.a.d.sendibm1.com/mk/cl/f/sh/WCPzyXJTZ7Ij90B1ZnOV0BpKWpsGkxmQ/C5-10fXFVNIC) lets you attach a summary to each chunk. When a query matches the summary, all related chunks are returned together, so the LLM gets fuller context.


You can add summaries manually, via API, or have an LLM generate them automatically.


[Read the blog →](https://b6usd.r.a.d.sendibm1.com/mk/cl/f/sh/WCPzyXJTZ7QWD2OHvssAlvz7fclO0fOY/pe4LbSTGe8Zw)
