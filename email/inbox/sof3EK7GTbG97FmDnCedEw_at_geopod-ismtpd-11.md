An agentic recap
----------------

* Last month, agent mode went GA ( https://github.blog/changelog/2025-04-03-github-copilot-in-vs-code-march-release-v1-99/?utm_source=github&utm_medium=email&utm_campaign=mpu_may ) in VS Code. This month, we previewed it in Codespaces ( https://github.blog/changelog/2025-04-11-vscode-copilot-agent-mode-in-codespaces/?utm_source=github&utm_medium=email&utm_campaign=mpu_may ).
* We’ve shipped GitHub Copilot code review ( https://github.blog/changelog/2025-04-04-copilot-code-review-now-generally-available/?utm_source=github&utm_medium=email&utm_campaign=mpu_may ) to GA, now supporting C, C++, Kotlin, and Swift ( https://github.blog/changelog/2025-04-23-github-copilot-code-review-now-supports-c-c-kotlin-and-swift/?utm_source=github&utm_medium=email&utm_campaign=mpu_may ) , and Copilot Autofix ( https://github.blog/changelog/2024-08-14-copilot-autofix-for-codeql-code-scanning-alerts-is-now-generally-available/?utm_source=github&utm_medium=email&utm_campaign=mpu_may ) with Security Campaigns in GA ( https://github.blog/changelog/2025-04-07-security-campaigns-are-now-generally-available-to-help-address-security-debt-at-scale/?utm_source=github&utm_medium=email&utm_campaign=mpu_may ).
* We made the GitHub MCP server ( https://github.blog/changelog/2025-04-04-github-mcp-server-public-preview/?utm_source=github&utm_medium=email&utm_campaign=mpu_may ) official, adding native support in VS Code. We also added remote MCP support and expanded agent capabilities for server-based tools and image generation in VS Code 1.100 ( https://code.visualstudio.com/updates/v1_100?utm_source=github&utm_medium=email&utm_campaign=mpu_may#_mcp-support-for-streamable-http ).

( https://github.blog/changelog/2025-05-08-openai-gpt-4-1-is-now-generally-available-in-github-copilot-as-the-new-default-model/?utm_source=github&utm_medium=email&utm_campaign=mpu_may )

More Copilot models, more control
---------------------------------

* *OpenAI’s GPT-4.1* is rolling out as the new default model ( https://github.blog/changelog/2025-05-08-openai-gpt-4-1-is-now-generally-available-in-github-copilot-as-the-new-default-model/?utm_source=github&utm_medium=email&utm_campaign=mpu_may ) for Copilot Chat, Edits, and agent mode.
* *Copilot Chat* and *code completion* are now available across VS Code, JetBrains, Eclipse, and XCode, with support for the latest cutting edge models, including Gemini 2.5 Pro ( https://github.blog/changelog/2025-04-11-copilot-chat-users-can-use-the-gemini-2-5-pro-model-in-public-preview/?utm_source=github&utm_medium=email&utm_campaign=mpu_may ) , OpenAI o3 and o4-mini ( https://github.blog/changelog/2025-04-16-openai-o3-and-o4-mini-are-now-available-in-public-preview-for-github-copilot-and-github-models/?utm_source=github&utm_medium=email&utm_campaign=mpu_may ) , and OpenAI GPT-4.1 ( https://github.blog/changelog/2025-04-14-openai-gpt-4-1-now-available-in-public-preview-for-github-copilot-and-github-models/?utm_source=github&utm_medium=email&utm_campaign=mpu_may ).
* Copilot Enterprise customers can now set custom instructions for all users ( https://github.blog/changelog/2025-04-17-organization-custom-instructions-now-available/?utm_source=github&utm_medium=email&utm_campaign=mpu_may ) in their organization, ensuring a consistent experience across teams.

( https://github.blog/changelog/2025-05-08-github-copilot-in-vs-code-april-release-v1-100/?utm_source=github&utm_medium=email&utm_campaign=mpu_may )

VS Code 1.100
-------------

The latest version of VS Code ( https://github.blog/changelog/2025-05-08-github-copilot-in-vs-code-april-release-v1-100/?utm_source=github&utm_medium=email&utm_campaign=mpu_may ) delivers several Copilot enhancements, including:

* Prompt caching and other enhancements will now lead to faster agent edits, especially in large files.
* Prompt and instructions files let you customize Copilot’s behavior with reusable prompts and coding guidelines.
* Select and attach UI elements in Copilot Chat with the built-in Simple Browser.
