We just released Claude Sonnet 4.6

Upgrades to Claude Code on desktop

New features for the Claude desktop app 
<https://code.claude.com/docs/en/desktop-quickstart?utm_content=inline_link> 
help you close the development loop without leaving Claude Code. You can now 
write, review, and verify code (plus monitor your open pull requests), all in 
the desktop app.

Learn more 
<https://claude.com/blog/preview-review-and-merge-with-claude-code?utm_content=inline_link>


 <https://code.claude.com/docs/en/fast-mode?utm_content=image_link> 
Fast mode for Opus 4.6 (research preview)

Use a 2.5x-faster version of Opus 4.6 at a higher cost per token with fast 
mode.

Toggle it on with /fast when you need speed for interactive work like rapid 
iteration or live debugging, and toggle it off when cost matters more than 
latency. Available for users with extra usage enabled.

Learn more <https://code.claude.com/docs/en/fast-mode?utm_content=inline_link>


 <https://code.claude.com/docs/en/agent-teams?utm_content=image_link> 
Agent teams (research preview)

Claude Code can now parallelize tasks across multiple agents that communicate 
peer-to-peer.

Note that running multiple agents may increase token usage proportionately. 
Agent teams are off by default and can be enabled in user settings.

Learn more 
<https://code.claude.com/docs/en/agent-teams?utm_content=inline_link>
