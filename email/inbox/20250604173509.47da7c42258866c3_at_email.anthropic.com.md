Hi there,



You can now access Claude Code with the Pro plan. The Pro plan gives you 
access to both Claude Code and Claude apps for a flat $20 monthly subscription.



The Pro plan is a great way to try out Claude Code, perfect for 1-2 hour 
coding sprints in smaller codebases.



Here’s what the Pro plan includes:

 * Claude Code access for your terminal-based coding workflows
 * Access to the Claude app where you can write, research, and create with 
Claude
 * Flexibility to upgrade to a Max plan for more usage and intensive Claude 
Code sprints
What you can expect:

Exact usage for Claude Code will vary 
<https://support.anthropic.com/en/articles/11145838-using-claude-code-with-your-max-plan>
 based on project complexity, codebase size, and auto-accept settings. With the 
Pro plan, you can send approximately 10-40 prompts with Claude Code every 5 
hours.



To get started with Claude Code on Pro:

 * Get Pro plan <https://www.anthropic.com/pricing>
 * Install Claude Code 
<https://docs.anthropic.com/en/docs/agents-and-tools/claude-code/overview#install-and-authenticate>
, or run `claude update && claude` if you already have it installed
 * Run `/login` in Claude Code and select the Pro plan option
