###Easy Streaming of Audit & Traffic Events to a Generic HTTP Endpoint


NetBird now supports [streaming](https://az5n7.r.ag.d.sendibm3.com/mk/cl/f/sh/1t6Af4OiGsDg0aYLa89CDFKOoFeboM/eHCWtK1sXqht/knowledge-hub/generic-http-endpoint-for-network-events-streaming) network activity events directly to any HTTP/S endpoint through its new Generic HTTP endpoint integration. This capability enables you to seamlessly integrate NetBird with your SIEM systems, observability platforms, custom applications, or third-party services. Ultimately, this feature enhances security and operational awareness by enabling immediate analysis, alerting, and auditing of network events within your organization's infrastructure tools.

 

Check out the [documentation](https://az5n7.r.ag.d.sendibm3.com/mk/cl/f/sh/1t6Af4OiGsF30j9clCQxi1VsxiaowY/r0eQWPslgRNb).

Simplifying Network Access Management With Port Ranges
------------------------------------------------------
======================================================
------------------------------------------------------


NetBird now supports [port ranges](https://az5n7.r.ag.d.sendibm3.com/mk/cl/f/sh/1t6Af4OiGsFVLRM39YWssHZiLXZYec/Qze7Xpkb1vxp) in access control policies. This new capability dramatically reduces the manual overhead of listing every single port for applications that require a wide range of them; simply define a port range (e.g. 8000-9000) in a single policy. This not only makes firewall and access rule configurations cleaner and more manageable but also improves security posture by making it easier to implement the principle of least privilege.

 

Check out the [documentation](https://az5n7.r.ag.d.sendibm3.com/mk/cl/f/sh/1t6Af4OiGsGQ0rktwGijCnhN7BX24k/6J1NdS0uKAnu).
