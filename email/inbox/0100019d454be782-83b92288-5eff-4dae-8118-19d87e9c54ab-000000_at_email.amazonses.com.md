Agno updates




GitHub Knowledge Sources: Production-ready authentication

 
Added support for GitHub App authentication alongside personal access tokens. This provides a more governed auth model with scoped
permissions, no personal credentials in the pipeline, and thread-safe token caching that automatically handles expiration.
 
The authentication model GitHub intended for production. View the cookbook →
[https://github.com/agno-agi/agno/blob/main/cookbook/07_knowledge/09_archive/cloud/github.py]
 
 
Google Slides Toolkit: Full control over Google Slides
 
Create, read, manage, and edit decks. Add slides with different layouts, insert tables, text boxes, images, and YouTube videos,
and reorder slides as needed.
 
These aren't export templates, this is an agent that turns a meeting brief into a finished deck. View the docs →
[https://docs.agno.com/tools/toolkits/others/google-slides?utm_source=newsletter&utm_medium=email&utm_campaign=agno-newsletter&utm_content=n-6&utm_term=google-slides]
 
 
Docling Reader: Stop building format-specific pipelines
 
Word docs, slide decks, spreadsheets, images, and even meeting recordings all flow through Docling Reader via a single interface.
And the structure is preserved so your agent retrieves context, not just text.
 
A unified ingestion layer that turns any document into knowledge. Learn more →
[https://docs.agno.com/knowledge/concepts/readers/docling-reader?utm_source=newsletter&utm_medium=email&utm_campaign=agno-newsletter&utm_content=n-6&utm_term=docling-reader]

PodcastBrain & AI Storyboard Generator
> built by Shubham Vedi

We want to highlight two of Shubham's March projects. PodcastBrain puts two agents with distinct personalities into a live
