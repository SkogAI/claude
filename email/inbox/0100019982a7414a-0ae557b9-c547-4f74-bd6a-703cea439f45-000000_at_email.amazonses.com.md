Dear API user,

We’re excited to launch the Perplexity Search API — giving developers direct access to the same real-time, high-quality web index
that powers Perplexity’s answers.
 
The Search API delivers factual, up-to-date results without the generative layer, so you decide exactly how to use the data inside
your own systems. Each query returns structured snippets ranked for relevance, letting you skip heavy pre-processing and plug
fresh web knowledge straight into your applications.
 
With Perplexity Search, you get:
 * Continuously refreshed index spanning hundreds of billions of pages

 * High accuracy at low latency, even at scale

 * Developer-friendly pricing at $5 per 1K requests

 
This is ideal for powering agents that need grounded web context, research tools that demand trust and freshness, and custom
products where you want full control over the output.
