**🌊 Windsurf Wave 11**
----------------------


 

Earlier this week, we announced that **Windsurf has joined forces with Cognition**, the team behind Devin. Since then, we’ve been ramping up on each other’s codebases and aligning on a shared mission: to build the best agentic IDE in the world.

Wave 11 is the first step in that journey. Here's what's new:


**Claude Sonnet 4 is back 👀**
-----------------------------


Claude Sonnet 4 is back via first party support from Anthropic. It’s available now at a reduced 2x credit rate for a limited time. Pro and Teams users get up to 250 requests each month.
 

Enjoy the boost. 🚀


**Voice Mode - Talk to Cascade (Literally)**
--------------------------------------------


As users have started asking Cascade to build full features, perform large refactors, and ship end-to-end PRs, the conversations have naturally gotten longer and more detailed. Sometimes, typing it all out is a bottleneck.

 

That’s why **Cascade now supports voice input**. Just click the mic icon and say what you want Cascade to do. No typing required.


**Checkpoints and @-mentions**
------------------------------


We’re making it easier to manage long and complex Cascade sessions:


- **Named Checkpoints:** Save conversation snapshots you can easily return to. 
- **@-Mention Conversations:** Reference previous Cascade chats by name to give new tasks full historical context. For example: *“Hey Cascade, add tests for the feature from @checkout-flow.”*


**Browser Flow, Fully Integrated**
----------------------------------


Wave 10 introduced the Windsurf Browser. In Wave 11, we’re taking it further.

Cascade can now **pull information directly from the browser** without you having to copy-paste anything. Think:


- Screenshots of UI bugs 
- Console logs 
- DOM elements 
You can also @-mention open tabs when chatting with Cascade.

This tightens the feedback loop between code and browser, especially for frontend development.

**Other Improvements**
----------------------

- **@-Mention Terminal:**Ask Cascade to reference specific terminal sessions. 
- **Planning Mode:**Now On by Default 
- **Auto-Continue:** Cascade will keep going automatically when it hits token limits. 
- **MCP OAuth + Streamable HTTP:** Easier, more secure authentication with MCP support and new transport protocols. 
- **Global .codeiumignore:** For enterprise teams working across multiple repos, set global ignore rules in ~/.codeium/.codeiumignore.
