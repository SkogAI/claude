Here’s what’s new


We’ve shipped several updates to Claude Code designed to get the busywork off 
your plate, so you can focus on the work that matters.



Claude Code channels (research preview)

Control your Claude Code session through select MCP servers, starting with 
Telegram and Discord. Claude reads the event and replies back through the same 
channel—so you can keep things moving while you’re not at your terminal.

Learn more <https://code.claude.com/docs/en/channels?utm_content=inline_link>



Scheduled tasks in Claude Code

Claude Code can now run recurring tasks 
<https://code.claude.com/docs/en/scheduled-tasks?utm_content=inline_link> on a 
schedule, both in the CLI or desktop app. Have Claude update documentation in a 
repo every week, or summarize open PRs each Monday so you know what needs 
review.

Learn more 
<https://code.claude.com/docs/en/scheduled-tasks?utm_content=inline_link>Try it 
on desktop <https://claude.ai/desktop/*?utm_content=inline_link>



Remote control for Claude Code

Start a session in your terminal, then control that session from the web or 
Claude mobile app usingremote control 
<https://code.claude.com/docs/en/remote-control>. Try it by running claude rc or
/remote-control.

Want to see how it works? Watch this tutorial video 
<https://claude.com/resources/tutorials/using-claude-code-remote-control?utm_content=inline_link>
.

Try it on iOS <https://apps.apple.com/us/app/claude-by-anthropic/id6473753684> 
or Android <https://play.google.com/store/apps/details?id=com.anthropic.claude>



Claude Opus 4.6 with 1M context

1M context <https://claude.com/blog/1m-context-ga?utm_content=inline_link> is 
now included in Max, Team, and Enterprise plans—no extra usage required. Select 
it with/model to get longer sessions, fewer compactions, and more room for 
context that matters.

Learn more <https://claude.com/blog/1m-context-ga?utm_content=inline_link>
