******************
Nuxt Studio Sunset
******************

Nuxt Studio is dead, long live Nuxt Studio

As announced when *NuxtLabs joined Vercel* , we are sunsetting the legacy *nuxt.studio* ( https://nuxt.studio ) platform.
This transition allows us to focus entirely on the new, *fully open-source, self-hosted Studio module* , built to offer the same experience without relying on an external SaaS.

---------------------------------------------------------------
*🌄 Why we are sunsetting* *nuxt.studio* ( https://nuxt.studio )
---------------------------------------------------------------

With the merger, Vercel’s goal is to *promote open source* and make all previously premium products *fully free* and open source. The same approach already taken with Nuxt UIPro and soon applied to NuxtHub.

For the NuxtLabs team, it means a lot, the opportunity to focus entirely on building tools that are *free, open source, and accessible to everyone*.

This is why the premium Studio platform will be discontinued.

------------------------------------------------------
*🚀 Meet the new Studio (Beta — official release soon)*
------------------------------------------------------

We rebuilt Studio from the ground up as a *Nuxt module* 👉 nuxt-studio ( https://github.com/nuxt-content/studio )
The Markdown editor has been open sourced by the Nuxt UI team and adapted on Studio side to be compatible with MDC syntax 👉 @nuxt/ui Editor component ( https://ui.nuxt.com/docs/components/editor )

You can install the new Studio module today and use it directly inside your project.

*Migration is extremely simple:*

* Follow the new module documentation (you can use Studio module and legacy platform at the same time).
* Upcoming versions of Nuxt Content will automatically remove all legacy Studio code but you can already remove the "preview" key in your Nuxt Content configuration if you don't need nuxt.studio ( https://nuxt.studio ) no more.

Your content remains safe, it has always lived in your Git repo.

*🚀* Roadmap: https://content.nuxt.com/docs/studio/roadmap
📚 Docs: https://content.nuxt.com/docs/studio/introduction
💚 Open source repository: https://github.com/nuxt-content/studio

------------
*📅 Timeline*
------------

*Now* → You can already migrate to the new module and cancel your subscription.
*End of December* → All nuxt.studio ( https://nuxt.studio ) subscriptions will end automatically.
*First week of January* → The legacy nuxt.studio ( https://nuxt.studio ) platform will be permanently shut down.
