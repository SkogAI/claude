What is the /compliance tool?

The /compliance tool takes over the key parts of /review,including ticket and security compliance, and combines them with new capabilities. It provides immediate, structured checks on every PR across security, ticket alignment, code duplication, and custom rules to automatically enforce organizational standards. By reducing overlap between tools and working alongside /improve for deeper code feedback, it delivers a simpler, more consistent workflow with stronger, more valuable checks
