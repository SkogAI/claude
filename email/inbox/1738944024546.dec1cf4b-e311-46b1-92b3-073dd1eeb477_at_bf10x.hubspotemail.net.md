🚀 What’s New in SmythOS?

✨ Form Preview – Test Before You Deploy

No more trial and error! Form Preview lets you see how your agent processes inputs before deployment. This feature presents your agent’s skills in an intuitive format, helping you:

- Test individual skill workflows instead of running the entire agent.
- Validate configurations in real time to prevent errors.
- Check input fields and ensure all required components are in place.
- Catch missing elements early—if an agent isn’t ready, Form Preview will alert you and disable “Run” until it’s properly configured.

🛠️ AgentLLM – Deploy AI Agents as OpenAI-compatible LLMs

Building AI-powered applications has never been easier! With AgentLLM, you can deploy SmythOS agents as OpenAI-compatible LLMs with just three lines of code.

- Passthrough Mode for Real-Time Debugging: See the intermediate steps of your agent’s execution, providing live updates and deeper insight into its reasoning.
- Multi-Version Support: Easily switch between agent versions, such as @dev, @prod, or numbered versions, for seamless testing and rollouts.

💬 Chat Message View Embodiment – AI Chat, Anywhere

Need an instant AI-powered chat experience for your app or website? Chat Message View Embodiment lets you export a ready-to-use chat UI directly from SmythOS and embed it in seconds.

- One-click export to generate an embeddable code snippet.
- Easily customize the chat UI’s design and behavior.
- Works seamlessly with your AI agents to handle conversations and queries.

🔗 One-Click Plugins – Google Sheets Integration

Tired of manually setting up API connections? SmythOS now offers a One-Click Plugin for Google Sheets, eliminating API key management!

- Instant setup: Just log into your Google account and connect in seconds.
- Seamless integration: Read, update, and manipulate data directly from Google Sheets.
- No coding required: Perfect for non-developers managing spreadsheets with AI automation.

🧠 New AI Models & Integrations – More Power, More Insights

We’re expanding SmythOS’s capabilities with new AI models and integrations to supercharge your applications:

- OpenAI o3 Mini: An optimized reasoning LLM for better agent performance.
- SerpStat: Advanced SEO and keyword research, now accessible within SmythOS agents.
- ScrapingBee: A seamless web scraping tool to extract real-time web data.

Now, your agents can analyze SEO trends, gather web insights, and interact with structured data in a fraction of the time!
