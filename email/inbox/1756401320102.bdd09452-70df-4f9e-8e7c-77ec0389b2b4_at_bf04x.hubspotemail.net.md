Hi Emil,

We've scheduled our second hands-on, virtual workshop for next Thursday, September 4th. This time, by popular demand, we will be walking everyone through exercises to build, run and deploy a flow in your favorite IDE (vs. Crew Studio).

CrewAI Flows is an event-based API that allows anyone who knows Python to build structured, complex agentic workflows that integrate LLMs, agents, crews with conditional logic and arbitrary code.

After a brief introduction to flows, we'll work through the process of building, running and tracing a real-world flow. Next, we'll break out into small groups to help address any issues. Finally, we'll regroup, and continue iterating on the flow.

Note: You will need a local environment to run Python code.
