Qwen3.5 from Alibaba is available on Ollama. It's a native vision-language model. Text and image understanding are built into a
single model:
 * Native multimodal: unlike previous generations, image input is built-in

 * 256K context: all models support 262,144 tokens, enough for most codebases or documents.

 * 201 languages: Expanded from 100+ in Qwen 3, covering languages and dialects

 * Thinking: All models support both thinking and non-thinking (fast)

 * Tool calling and agents: Native tool calling and strong agentic capabilities across all sizes

 


Get started

To run Qwen3.5 with tools such as Claude Code or OpenClaw, use Ollama’s cloud to run the largest, most powerful 397B version of
the model:


ollama launch claude --model qwen3.5:cloud

ollama launch openclaw --model qwen3.5:cloud



Available sizes

Qwen 3.5 is available in large, medium and small sizes:
 
Large
The full 397B-A17B model is available on Ollama’s cloud, with 17B active parameters per token and good performance across
reasoning, coding, and vision tasks.


ollama run qwen3.5:397b-cloud
 
Medium
The 27B is a dense model—all parameters are active on every pass, giving it the highest per-token reasoning density at this tier.
 
The 35B-A3B activates only 3B parameters per token, making it faster while matching the performance of much larger
previous-generation models.
 
The 122B-A10B sits in between, with 10B active parameters and strong tool use performance.
 
ollama run qwen3.5:27b
 
ollama run qwen3.5:35b-a3b
 
ollama run qwen3.5:122b-a10b
 
Small
The 9B model is the default when running ollama run qwen3.5. It fits comfortably on a 16GB GPU and supports text, image input,
thinking, and tool calling. 4b, 2b, and 0.8b models are also available.
 
ollama run qwen3.5:9b
 
ollama run qwen3.5:4b
 
ollama run qwen3.5:2b
 
ollama run qwen3.5:0.8b
