Visual Studio Code now directly integrates with Ollama via GitHub Copilot.
 

Download Ollama [https://ollama.com/download]

 
If you have Ollama installed, any local or cloud model from Ollama can be selected for use within Visual Studio Code.
 




GitHub Copilot Chat agents

Ollama's models work out-of-the-box with features from Copilot Chat agents.


Agent mode

In agent mode, Copilot will use Ollama models (local or Ollama's cloud) to complete end-to-end tasks: determining which files to
modify, suggesting code changes, running terminal commands, and iterating until the task is complete.

Try prompts like:
 

  - "Run the tests and fix any failures"                                          
  - "Generate unit tests for this file"
  - "Update the README with the new API changes"


Plan mode

In plan mode, Copilot will use Ollama models to create detailed implementation plans before prompting you to execute them.
 


Ask mode

Ask mode is optimized for answering questions about the codebase, coding or general concepts. Use ask mode when you want to
understand how something works, explore ideas, or get help with coding tasks.
