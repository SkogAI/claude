###**NetBird Python Client (Unofficial)**
NetBird is strengthened by community contributions. We’d like to highlight the NetBird Python Client library created by [Bhushan Rane](https://az5n7.r.ag.d.sendibm3.com/mk/cl/f/sh/1t6Af4OiGsEag0xCMqL2XlS3Ztc5EU/eVvH1Wmg5t8V) that provides complete access to all NetBird API resources with a simple, intuitive UI. This client follows the same upstream schemas as the official NetBird REST APIs, ensuring full compatibility and consistency with the NetBird ecosystem.

###[Traffic Events Logging](https://az5n7.r.ag.d.sendibm3.com/mk/cl/f/sh/1t6Af4OiGsHn10MB7L0UhZsrGeTFCw/FCLiz4qGpkgk)


NetBird's Traffic Events Logging *(in Beta)* provides comprehensive insights into network connections, enabling you to see both successful and blocked peer-to-peer and peer-to-network resource connections. Also, you can limit logging to specific groups of peers for more focused analysis. This data can be streamed to Security Information and Event Management (SIEM) systems, and logs are retained for seven days.

[Learn more](https://az5n7.r.ag.d.sendibm3.com/mk/cl/f/sh/7nVU1aA2nfsZpYhz5TNontOL9tnQTm2/Gpm5q1VsRIVI).
