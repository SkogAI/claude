We’ve got a Mega Launch Week coming up Dec 2-6, a seamless partnership with Clickhouse, and more. Let’s get started:

Mega Launch Week (Dec 2-6)
We’re kicking off Mega Launch Week during the week of December 2-6. What makes it “Mega”? Well, this time around we’ve asked our devtool friends to join in on the fun. They’ll be hosting their own Launch Weeks in parallel during the same week for a bigger bang. If you’d like to get in on the action drop us a message here: https://supabase.link/mega-lw-tweet
Mega Launch Week (https://supabase.link/launchweek-readme) | Supabase Launch Week Meetups (https://supabase.link/megalw-meetups)

Supabase x Clickhouse Partnership
Postgres and ClickHouse are becoming the “default data stack” for AI companies. They're two sides of the same data coin: one shines when it comes to transactional data while the other excels at analytics. As part of the partnership, both Supabase and Clickhouse have added in functionality to make interoperability between the two more seamless.
Announcement: https://supabase.link/supabase-clickhouse

Edge Functions: Write in Pure JavaScript and Import private NPM Packages
If you much prefer JavaScript over TypeScript, you can write your Edge Functions in JavaScript. Specifically you can use .js, .jsx, or .mjs files as your Function entrypoint. That’s not all, now you can import NPM packages from private registries and it’s a good option if you want to share code and types between your frontend and Edge Functions.
JavaScript Functions (https://supabase.link/edgefunctions-in-js) | Private NPM Packages (https://supabase.link/private-npm-packages)

Auth: Email Template Spam Checker
We’ve added a spam checker via self-hosting Apache SpamAssassin that checks your Supabase Auth custom email templates for spam so you can make necessary adjustments to ensure email deliverability to your users.
Changelog: https://supabase.link/auth-email-checker

Studio: Table Editor Performance Improvements
Over the last few weeks we focused on making our Table Editor faster and smoother by shortening both perceived and actual load times as you navigate by optimizing queries and by prefetching.
Changelog: https://supabase.link/table-editor-performance

Realtime: Connection Persists in Background Tabs
If you’ve used Realtime in a client-side app, you may have run into the issue where Realtime stops working when the app is in a background tab. We’ve fixed that by moving our interval heartbeat mechanism to a Web Worker.
Pull Request: https://supabase.link/realtime-persist-connection
