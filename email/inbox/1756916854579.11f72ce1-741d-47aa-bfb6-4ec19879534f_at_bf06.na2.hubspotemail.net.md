💻 OpenSWE: Async Coding Agent

Open SWE is an open-source, asynchronous coding agent that acts like a true teammate. Built on LangGraph, OpenSWE connects directly to your GitHub repo, plans and reviews its own work, and goes to fix bugs or implement features. It is cloud-hosted, extensible, and designed to make long-running coding agents reliable and collaborative.
