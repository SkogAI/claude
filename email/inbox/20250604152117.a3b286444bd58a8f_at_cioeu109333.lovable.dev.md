Hey,

In the last 2 weeks we've done a bunch of improvements to Lovable!

This is what's new:

---------------
Security Review
---------------

Security Review

Our new security reviewer uses AI to analyze your app for security vulnerabilities and recommends a plan to address them.

--------------------------------------
Warnings if you put an API key in chat
--------------------------------------

Warnings if you put an API key in chat

Previously, people would sometimes ask Lovable to hardcode API secrets into their app, which posed a major security risk as those secrets could be easily leaked. We now block around 1200 private API keys from being directly inserted in code per day.

All API keys should be stored as a secret in Supabase via our special in-chat form.

----------------------------------
Filtering of own and team projects
----------------------------------

Filtering of own and team projects

Improved project discoverability with new filtering options to easily find and manage your projects.

------------------------------------
Categorization of community projects
------------------------------------

Categorization of community projects

Categorization to community projects to make it easier to discover projects by the community.

You can now also see the amount of remixes each project has.

------------------------
Visualize back-end flows
------------------------

Visualize back-end flows

Chat mode can now render Mermaid diagrams to help visualize your app's back-end logic and structure (ask for it to chart things to trigger this).

------------------
Other Improvements
------------------

* Added voice mode functionality - simply speak your ideas and have them automatically converted to text for faster development.
* Auto-configure redirect URLs for Supabase Auth integration - no more manual configuration needed for a more seamless authentication setup.
* Added in-product feedback system - click the thumbs down button when edits cause issues to help us improve Lovable's AI for everyone.
* Introduced smart nudges to guide you to Chat Mode when debugging issues that need conversational help.
