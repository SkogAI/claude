Product Updates
***************

Toolhouse AI Agents now run in n8n

Our new n8n Community Node is here. You can now seamlessly integrate Toolhouse Agents into your n8n workflows in seconds.

Your agents come pre-loaded with Memory, MCP Servers (Model Context Protocol), Code Interpreter, RAG capabilities, and more.

All you need is a simple prompt describing what your agent should accomplish and your API key.
It's automation made simple.

Try the n8n integration ( https://docs.toolhouse.ai/toolhouse/integrations/n8n-integration )

Toolhouse now integrates with Zapier

Our new Zapier integration is live! You can now connect Toolhouse Agents to thousands of apps like Slack, Gmail, Airtable, and HubSpot without writing code.

Trigger agents from Zapier events, pass in data, and route responses back into your workflows in just a few clicks.

All you need is your API key and a Toolhouse agent, and automation has never been easier.

Check out our Zapier integration here ( https://docs.toolhouse.ai/toolhouse/integrations/zapier-integration )

Your AI Agents Can Now Talk to Supabase in Minutes

Say goodbye to backend complexity! Our free Supabase MCP server lets your AI agents read from and write to your Supabase database without writing a single line of backend code. Connect your agents to your data effortlessly.

Learn how to connect AI to Supabase ( https://docs.toolhouse.ai/toolhouse/integrations/supabase-integration )

Scaffold frontends with `th vibe` in the CLI!

Our new CLI command is here! With th vibe, you can instantly generate a custom prompt that helps you scaffold frontends for your agents using platforms like Lovable, v0, or Bolt.

No more starting from scratch,  just run th vibe on any deployed YAML file, and you’ll get a ready to use prompt tailored for your frontend.

Building and deploying agents has never been smoother.

Learn more about th vibe here ( https://docs.toolhouse.ai/toolhouse/advanced-concepts/generate-a-vibe-coding-prompt-with-toolhouse-cli )

Toolhouse MCP Client: One Link, Thousands of Possibilities

We're bringing MCP support to every Toolhouse agent! Now you can connect to thousands of remote MCP servers with a single link, instantly plugging your agents into countless services and platforms. Find any remote MCP server link, add it to your agent, and you're ready to go.

Try out an MCP enabled agent ( https://app.toolhouse.ai/agents/new?message=build+an+agent+that+connects+to+the+cloudflare+MCP+server+https%3A%2F%2Fdocs.mcp.cloudflare.com%2Fsse+to+answer+user+questions+about+cloudflare.+assume+the+user+is+not+technical )
