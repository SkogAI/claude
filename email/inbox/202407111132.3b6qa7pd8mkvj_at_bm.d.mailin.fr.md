Welcome to our November newsletter! This month, we’ve focused on further improvements, feature highlights, and important updates on accessible security options. Here’s a quick look at what’s new:

Setup Key Improvements

We’ve introduced several updates to improve setup key management:

- Flexible Expiration: Users can set custom expiration dates for setup keys without limitations, including non-expiring keys.
- Key Deletion Option: You can now delete setup keys, making it easier to manage expired or revoked keys and keep your account clean. https://az5n7.r.a.d.sendibm1.com/mk/cl/f/sh/1t6Af4OiGsE8LIklyUF7NVOEC4dLWQ/9btkVTmZSeTg

Billing Admin Role

We added a new "Billing Admin" role that allows designated users to manage Plans & Billing. This feature provides flexibility in managing account resources while ensuring only authorized team members have billing access.

Security for All: Free SSO and MFA

We’re happy to let you know that MFA is now free for all users, including in our cloud-hosted version. Robust security features like Single Sign-On (SSO) and Multi-Factor Authentication (MFA) are more essential than ever, and we’re making them accessible to everyone - no paywalls, no restrictions.

Why It Matters: In a time when key security features are often locked behind premium paywalls, many users are left with limited protection. By making MFA free, NetBird aims to change that. Security shouldn’t be a luxury; it’s a necessity.

Seamless Integration: NetBird’s integration with popular Identity Providers (IdPs) like Google, Azure, and Okta makes MFA setup straightforward. For those without an IdP, enabling MFA is easy through our web UI.

Read more about our commitment in the latest article, “[Security for All - SSO and MFA for Free](https://az5n7.r.a.d.sendibm1.com/mk/cl/f/sh/1t6Af4OiGsEag0xCMqL2XlS3Ztc5EU/AN7Mmkn7C_IS).”
