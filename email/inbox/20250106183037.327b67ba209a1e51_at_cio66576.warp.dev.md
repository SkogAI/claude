Happy New Year from Warp! Here is a summary of our biggest product updates from 2024.

Our AI features got a BIG upgrade.

 

Agent Mode ( https://www.warp.dev/ai ) can understand plain English queries to suggest and execute commands. It can even generate code diffs for tasks like fixing compiler errors or modifying code.

Active AI ( https://docs.warp.dev/features/warp-ai/active-ai ) prompts you what to ask based on your errors/input/output, and even suggests the next command to run. Watch a video ( https://x.com/warpdotdev/status/1868720420553404529 ) of Active AI in action.

You can choose your own AI model ( https://docs.warp.dev/features/warp-ai/agent-mode#how-to-choose-your-model-in-agent-mode ), whether it’s GPT-4o, Claude 3.5 Sonnet, or Claude 3.5 Haiku.

Learn about Warp's full AI capabilities ( https://docs.warp.dev/features/warp-ai/agent-mode )
We lifted our login requirements.

 

You can now use Warp without signing up for an account or logging in first. This gives you access to all of our core features. You can decide whether you want to authenticate to unlock more AI and team features when you’re ready.

Read about why we lifted Warp's login requirement ( https://www.warp.dev/blog/lifting-login-requirement )
Warp has a whole new suite of collaboration features

 

Session sharing ( https://www.warp.dev/session-sharing ) lets you collaborate with other developers in a Warp terminal session for pair programming, incident response, or live onboarding. Check out a video ( https://x.com/warpdotdev/status/1866884152102097145 ) here.

Warp Drive on the Web ( https://docs.warp.dev/features/warp-drive/warp-drive-on-the-web ) allows you to access Warp Drive objects and shared sessions from the browser, even without a Warp account.

Notebooks ( https://docs.warp.dev/features/warp-drive/notebooks ) are interactive run books that combine markdown, code, and runnable shell snippets so you can run team documentation without leaving Warp.

Environment variables ( https://x.com/warpdotdev/status/1839744911178097082 ) let you load your team’s variables into your terminal session and even dynamically reference secrets from external password managers like 1Password and LastPass.

Learn more about Warp's collaboration features ( https://www.warp.dev/warp-drive )
Warp on Linux AND Windows wen?

 

In February, we released Warp for Linux ( https://app.warp.dev/get_warp )! You can now install Warp on most Linux distributions, (e.g. Ubuntu, Fedora, Arch Linux, Red Hat), including their ARM64 builds.

We’re also thrilled to share that Warp for Windows is just around the corner ( https://x.com/warpdotdev/status/1866498114577739873 ), and we’re currently onboarding users from the Windows waitlist into our trusted testers program.

Sign up for the Windows waitlist ( https://www.warp.dev/windows-terminal )
