Greater access points and more models for Copilot
-------------------------------------------------

* Copilot CLI launched in public preview: ( https://github.blog/changelog/2025-09-25-github-copilot-cli-is-now-in-public-preview/?utm_source=github&utm_medium=email&utm_campaign=agentic-copilot-cli-launch-2025 ) Interact with Copilot right in your terminal so your AI-coding agent has full access without any context switching.
* *Claude Sonnet 4.5 ( https://github.blog/changelog/2025-09-29-anthropic-claude-sonnet-4-5-is-in-public-preview-for-github-copilot/?utm_source=github&utm_medium=email&utm_campaign=mpu_october2025 ) and Claude Opus 4.1 ( https://github.blog/changelog/2025-09-23-claude-opus-4-1-is-now-generally-available-in-github-copilot/?utm_source=github&utm_medium=email&utm_campaign=mpu_october2025 ) are now available:* Use the latest Claude models for your prompts, or spin up a coding agent request with Claude Sonnet 4.5 ( https://github.blog/changelog/2025-09-30-anthropic-claude-sonnet-4-5-is-in-public-preview-for-copilot-coding-agent/?utm_source=github&utm_medium=email&utm_campaign=mpu_october2025 ).
* The OpenAI GPT-5 models are here: ( https://github.blog/changelog/2025-09-09-openai-gpt-5-and-gpt-5-mini-are-now-generally-available-in-github-copilot/?utm_source=github&utm_medium=email&utm_campaign=mpu_october2025 ) This suite of models includes GPT-5, GPT-5 mini, and GPT-5-Codex ( https://github.blog/changelog/2025-09-23-openai-gpt-5-codex-is-rolling-out-in-public-preview-for-github-copilot/?utm_source=github&utm_medium=email&utm_campaign=mpu_october2025 ).
* Use Copilot code review in Visual Studio and JetBrains IDEs: ( https://github.blog/changelog/2025-09-18-copilot-code-review-now-in-jetbrains-ides-and-visual-studio/?utm_source=github&utm_medium=email&utm_campaign=mpu_october2025 ) Support for Copilot code review now exists in these IDEs, reaching parity with Xcode and VS Code.

( https://github.blog/changelog/2025-09-25-copilot-coding-agent-is-now-generally-available/?utm_source=github&utm_medium=email&utm_campaign=mpu_october2025 )

Say hello to your new programmer
--------------------------------

* Copilot coding agent is now generally available: ( https://github.blog/changelog/2025-09-25-copilot-coding-agent-is-now-generally-available/?utm_source=github&utm_medium=email&utm_campaign=mpu_october2025 ) Assign an issue or task to Copilot, and have it work in the background to create a pull request for you to review. Use it to get a headstart on some coding tasks or to address your backlog.
* Assign work to Copilot coding agent through Azure Boards: ( https://github.blog/changelog/2025-09-18-assign-azure-boards-work-items-to-copilot-coding-agent-in-public-preview/?utm_source=github&utm_medium=email&utm_campaign=mpu_october2025 ) Hand off work items to Copilot directly through Azure Boards.
* Give Copilot work from Microsoft Teams: ( https://github.blog/changelog/2025-09-19-work-with-copilot-coding-agent-in-microsoft-teams/?utm_source=github&utm_medium=email&utm_campaign=mpu_october2025 ) Ask Copilot to create a pull request by mentioning @GitHub it in a Teams conversation with a prompt.
* GitHub Spark available in public preview for Copilot Enterprise plans: ( https://github.blog/changelog/2025-09-30-github-spark-in-public-preview-for-copilot-enterprise-subscribers/?utm_source=github&utm_medium=email&utm_campaign=mpu_october2025 ) Transform your ideas into full stack intelligent apps.

( https://github.blog/changelog/2025-09-16-github-mcp-registry-the-fastest-way-to-discover-ai-tools/?utm_source=github&utm_medium=email&utm_campaign=mpu_october2025 )

Greater insight = greater confidence
------------------------------------

* Discover more with the GitHub MCP Registry: ( https://github.blog/changelog/2025-09-16-github-mcp-registry-the-fastest-way-to-discover-ai-tools/?utm_source=github&utm_medium=email&utm_campaign=mpu_october2025 ) Find the right MCP server to suit your needs by browsing this curated collection.
* Tracking GitHub Enterprise Cloud (GHEC) usage by organization: ( https://github.blog/changelog/2025-09-22-visualize-metered-usage-by-organization-in-github-enterprise-cloud/?utm_source=github&utm_medium=email&utm_campaign=mpu_october2025 ) Get more detailed visibility into which organization is using each GHEC license.
* A new dashboard shows your premium request analytics: ( https://github.blog/changelog/2025-09-30-premium-requests-analytics-page-is-now-generally-available/?utm_source=github&utm_medium=email&utm_campaign=mpu_october2025 ) Increase your awareness for when and where requests are being used.
* Assign code scanning and secret scanning alerts: ( https://github.blog/changelog/2025-09-23-accelerate-remediation-with-security-campaigns-and-assignable-alerts-for-code-scanning-and-secret-scanning/?utm_source=github&utm_medium=email&utm_campaign=mpu_october2025 ) Assign alerts directly to specific users and create new campaign types for large-scale remediation efforts.
