New Filter Sidebar

We've redesigned the filtering experience in Langfuse tables [https://langfuse.com/changelog/2025-11-03-filter-sidebar-for-tables]
to be more intuitive, powerful, and fast. The new sidebar lets you instantly filter by column values with a single click.
To help you spot trends, it also shows the number of occurrences for each value within your selected time range.

https://langfuse.com/changelog/2025-11-03-filter-sidebar-for-tables

 


Filtering for Traces and Observations API

For our power users and programmatic integrations, we're bringing a new level of precision to the traces and observations API.
You can now use a new filter parameter [https://langfuse.com/] to pass complex, JSON-encoded filter objects, enabling you to build
queries for more precise data retrieval. This new method allows you to query any field, including datetime, string, number, and
array types.

https://langfuse.com/changelog/2025-11-03-advanced-filtering-traces-and-observations-api
