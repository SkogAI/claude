Hi there,



Great news: Claude Code is now available on the Max plan. Now, you can use two 
powerful AI products — Claude Code and Claude apps, in one simple subscription.



Here’s what the Max plan includes:

 * Claude Code for your terminal-based coding workflows
 * Claude apps for writing, research, and analysis
 * Clear monthly billing — no more tracking per-token costs
What you can expect:

With the Max plan, your usage limits are shared across both Claude Code and 
Claude apps.



Expected usage for Claude Code will vary based on project complexity, codebase 
size, and auto-accept settings.Learn more 
<https://support.anthropic.com/en/articles/11145838-using-claude-code-with-your-max-plan>
.

 * On the Max plan ($100/month), you can send approximately 50-200 prompts 
with Claude Code every 5 hours
 * On the Max plan ($200/month), you can send approximately 200-800 prompts 
with Claude Code every 5 hours
To get started with Claude Code on Max:

 * Get Max plan <http://claude.ai/upgrade>
 * Install Claude Code 
<https://docs.anthropic.com/en/docs/agents-and-tools/claude-code/overview#install-and-authenticate>
, or run `claude update && claude` if you already have it installed
 * Run `/login` in Claude Code and select the Max plan option
We’re excited to see what you build next with the flexibility of the Max plan!
