What’s New in Zenflow & Zencoder

Subagents, CLI orchestration, and a redesigned Zenflow app

We usually share product updates on a monthly cadence, but this release couldn’t wait. Here’s what shipped.

Orchestrate CLIs, Subagents

You can now orchestrate CLIs- not just models - within Zenflow and Zencoder IDE plugins. For example, you can review Claude Code output with Codex, or vice versa.

Subagents are lightweight agents that run as subprocesses of your current session. Each subagent can use its own model, tools, or CLI, and returns results to the main agent. This works across both Zenflow and the IDE plugins, so you can run tools like Codex, Claude Code, or Gemini CLI in parallel—compare outputs, or have one review another.

Because subagents don’t carry full session context, they produce cleaner results for focused tasks like planning, code review, or research.

Use them on demand via /skillX, automatically during planning and exploration, or from the IDE with “Get second opinion.”

Zenflow Desktop App

Redesigned layout

Zenflow now uses a three-panel layout: header bar, chat and content panels side by side, and a collapsible tools panel for Steps, Changes, and Files. Nothing buried in tabs.

File attachments

Drag, paste, or pick any file into chat messages and task descriptions. Images preview inline on click.

AI-executed verification

Verification scripts run through the agent. It reads the output, iterates on failures, and fixes issues without manual intervention.

Quote-to-input

Select text in an agent response and quote it directly into your chat input. Useful when iterating on a specific part of the output.

Also shipped:

Unread indicators when an agent finishes. Sticky group-by tabs in the sidebar. Task status icons in "By project" view. Launch on startup (macOS and Windows). Smaller terminal font. Scroll-to-bottom in chat. Improved diff viewer colors. Lastly, the workflow steps are now under "todo" (top right).
