The Google Shopping Filters API (
https://click.kit-mail1.com/k0uo750e2wf6h5l0qxqhlhokz6777i8h62pl0/g3hnh5hmewveqpb3/aHR0cHM6Ly93d3cuc2VhcmNoYXBpLmlvL2RvY3MvZ29vZ2xlLXNob3BwaW5nLWZpbHRlcnM=
) returns all available filters for a shopping query, including
those hidden in the standard Google Shopping API. It provides:

* Filter Results – Organized by type (e.g., Category, Brand,
Price).
* Options – Each filter option contains a label (e.g.,
PlayStation Controllers, PlayStation Games, Video Games) along
with a shoprs token that can be used to refine the search.

JSON keys are kept the same as in the Google Shopping API for
seamless integration.
