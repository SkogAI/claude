**Hello NetBird Community!**

 

We've kicked off 2025 with major platform improvements focused on performance, security, and flexibility. 

 


Product Updates
---------------
 


###Client and Management 
###----------------------
Besides the many quality-of-life improvements, we’ve also added some exciting new features. 

 


- Introduced QUIC protocol support in the client and relay server, bringing better connection reliability and performance for relayed connections. 
- Added MySQL support to the management service - perfect for teams preferring MySQL as their database backend. 
- Implemented Rootless Container support for clients running Netstack mode. 
 


###Dashboard 
###----------
The Dashboard got some love too this past month with several significant improvements.
 


- Improved masquerade states, resource searching, and improved network setup flow 
- Improved confirmation dialogues, maintained sort order consistency, and fixed group badge display issues 
- Resource counters for groups, installation options after setup key creation, and toggles for resources and routing peers
