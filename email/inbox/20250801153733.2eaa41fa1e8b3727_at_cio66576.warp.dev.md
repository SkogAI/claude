It’s been a little over one month since Warp launched the first ever Agentic Development Environment with Warp 2.0 ( https://www.warp.dev/blog/reimagining-coding-agentic-development-environment ). Read on to see what’s happened since:

*******************
Coding improvements
*******************

Warp now has a richer file viewing and editing experience— all within the app.

We now support: Searching files, tabbed file viewer, find and replace, syntax highlighting for more frameworks, direct links to lines of code, code blocks as context, and collapsible blocks for less cluttered UI
