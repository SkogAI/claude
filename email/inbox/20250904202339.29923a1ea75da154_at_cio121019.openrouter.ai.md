Hi there,We launched the first-ever image model on OpenRouter: Gemini 2.5 Flash Image Preview ( https://openrouter.ai/google/gemini-2.5-flash-image-preview ).

This model combines the intelligence of LLMs with the visual quality of diffusion models, unlocking workflows like:

* State of the art consistency and prompt adherence
* Creating logo iterations in bulk
* Generating multiple images in a single call
In addition to new models, we have some great new features now generally
available for all OpenRouter customers:* Analytics API ( https://openrouter.ai/docs/api-reference/analytics/get-activity ) - Programmatically retrieve usage activity by day, by endpoint. Great for automating accounting or ingesting into a data warehouse for reporting.
* Zero Data Retention (ZDR) ( https://openrouter.ai/docs/features/zdr ) - OpenRouter itself has always been ZDR, but we're now tracking data retention status for every single model and provider. We've also introduced a one-click setting to restrict your account to ZDR endpoints for organizations that require such a data policy.
