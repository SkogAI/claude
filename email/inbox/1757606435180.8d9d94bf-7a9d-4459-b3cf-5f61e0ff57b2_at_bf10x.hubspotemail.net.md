Hi there, 👋

We're excited to announce a simpler and more secure process for authenticating new components in SmythOS. Over the past three months, we've upgraded the SmythOS Vault to support a centralized OAuth process, which removes the need to manage authentication for each tool you connect.

What does this mean for you?

- Migration is highly recommended. We’ve put together a short migration video (https://mail.inkforall.com/e3t/Ctc/UB+113/d2r66n04/VWs2Sg6_QnTMW3g_s-h3B6FzwW6wcfC_5CkxtQN3Z5r9H3prCCW7lCdLW6lZ3l6W1FWSnS1LjqPPW8P88wH3mvVqZW6QRH4Z8hrtjTW7CTvjN4rwRwWW899Y-Q9d5G-wW20RsXN8hpN7fW55VHWZ7m94XtW13Jpjm8jkfFlW6LfF4Z4gG5R6W6ZCcXv1h98cVW8-n4B87BLT4TW4hB36M1Tz0flW69gJ3F3_B0gbW5DpN8p7hQ_4bN1WHlw9jnGYNW1k1bcM693b1RMQkw8G7c4s8W2TdfvH22hDZLW61gKT53_z1T1W4tl8pV6BnbsFM119WFn6VbzW3H5LZZ2fr-MgW7fK_gY1NdNqbW2vYPwx8F5Brjf4zs9Zv04 ) (less than 2 minutes) to walk you through the process. Your existing agents will continue to run without any disruption, but if you want to make changes to them, you’ll need to migrate to the new component first.
- Faster setup moving forward. All new connections now use a centralized authentication flow, making it quicker to add tools and simpler to manage them over time.

Our goal is to give you a more secure and efficient way to build, so you can spend less time on credentials and more time creating. If you have any questions, please reach out to us at support@smythos.com (mailto:support@smythos.com) .

Thank you for being a part of the SmythOS community!
