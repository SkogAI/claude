The two big updates this week for AI builders:

1. Claude Code now has custom subagents you can use for specific tasks (each with their own full context window)
2. From next month, Claude Code will have [weekly rate limits](https://x.com/iannuttall/status/1949900781236473910) (which should only affect \~5% of users apparently)

Now I use subagents a lot, usually to speed up discovery and backfilling context by asking Claude to review my codebase to gain an understanding of it before building out a new feature.

It's kind of ironic that using subagents will burn through the weekly limits quicker so will be interesting to see how this works in practice.

Although I show a lot of examples of spinning up multiple agents, subagents, and complicated setups, my preferred vibe coding flow for deep work is a lot slower and more controlled:

- Spend 30 minutes chatting with Claude (usually in the desktop app) to understand what I want to build and what packages I need to use
- Create a full and detailed step-by-step plan in a Markdown file
- In Claude Code, I work through the plan step-by-step. I don't give the entire file, because inevitably there are problems that I need to solve along the way, and I prefer to work slowly so I understand what is getting built
- I use my [Claude Sessions custom slash commands](https://github.com/iannuttall/claude-sessions) to log everything I build, every issue I fix, every decision I make, so I can refer back to it and use as context later

Outside of crazy demos with 5+ agents running in custom git worktrees to see what is possible, this is what I truly consider "vibe coding".

Caring about what you build enough to be part of the workflow and not just letting AI think and act for you is a pre-requisite for success.

The bonus is that since it's a slower process I really don't think these weekly rate limits will affect me all that much on the 20x Max plan.

If they do, well I can always buy another one (or wait for Anthropic to bring out the 50x plan!).

What do you think to the rate limit announcements? Did I miss anything important over the weekend? Hit reply and let me know.
