Agno updates




Learning machines: build agents that learn from experience


Learning Machines continuously integrate information from their environment and improve over time—across users, sessions, and
tasks.


These aren’t agents with memory, this is the next evolution. Learn more about Learning Machines →
[https://docs.agno.com/learning/overview#learning-machine?utm_source=newsletter&utm_medium=email&utm_campaign=agno-newsletter&utm_content=n-4&utm_term=learning-machines]
 


Agent Skills: extend what your agents can do



Anthropic-compatible Agent Skills provide domain expertise on demand through reusable knowledge packages that can be shared across
agents. A progressive, lazy-loaded discovery model ensures each agent pulls in only the skills it needs, when it needs them.
 
Write once, reuse everywhere. Explore Agent Skills →
[https://docs.agno.com/skills/overview?utm_source=newsletter&utm_medium=email&utm_campaign=agno-newsletter&utm_content=n-4&utm_term=skills]
 


GitHub integration: unlock private repositories


With first-class support for private repositories, agents can now access GitHub repositories and perform tasks such as listing
open pull requests, issues, and more.
 
Your internal knowledge, securely accessible to agents. Learn more about the GitHub integration →
[https://docs.agno.com/tools/toolkits/others/github#github?utm_source=newsletter&utm_medium=email&utm_campaign=agno-newsletter&utm_content=n-4&utm_term=github-integration]
