More models, more options
-------------------------

* OpenAI GPT-5 is here: ( https://github.blog/changelog/2025-08-07-openai-gpt-5-is-now-in-public-preview-for-github-copilot/?utm_source=github&utm_medium=email&utm_campaign=mpu_august2025 ) Available for public preview in *Chat* , *Ask* , or *Agent* mode, the model is available in VS Code, on github.com ( http://Github.com?utm_source=github&utm_medium=email&utm_campaign=mpu_august2025 ) , and on GitHub mobile.
* Anthropic Claude Opus 4.1 is in public preview for Copilot Enterprise users: ( https://github.blog/changelog/2025-08-05-anthropic-claude-opus-4-1-is-now-in-public-preview-in-github-copilot/?utm_source=github&utm_medium=email&utm_campaign=mpu_august2025 ) Now available in Copilot Chat, Claude Opus 4.1 succeeds Opus 4 as Anthropic’s most capable model to support extended thinking, tool use, and logical summaries.

( https://github.blog/changelog/2025-07-02-agents-page-for-copilot-coding-agent-in-public-preview/?utm_source=github&utm_medium=email&utm_campaign=mpu_august2025 )

Copilot coding agent is easier to access and more customizable to meet your needs
---------------------------------------------------------------------------------

* Simplified costs: ( https://github.blog/changelog/2025-07-10-github-copilot-coding-agent-now-uses-one-premium-request-per-session/?utm_source=github&utm_medium=email&utm_campaign=mpu_august2025 ) The agent now uses a single Copilot premium request ( https://docs.github.com/copilot/concepts/billing/copilot-requests#what-are-premium-requests?utm_source=github&utm_medium=email&utm_campaign=mpu_august2025 ) per session, regardless of how complex the task is. Know the cost before starting a request, and complete more tasks within your monthly request limit!
* Start from (almost) anywhere: ( https://github.blog/changelog/2025-07-02-agents-page-for-copilot-coding-agent-in-public-preview/?utm_source=github&utm_medium=email&utm_campaign=mpu_august2025 ) Kick off Copilot coding agent requests from the new *Agents* page, VS Code ( https://github.blog/changelog/2025-07-14-start-and-track-github-copilot-coding-agent-sessions-from-visual-studio-code/?utm_source=github&utm_medium=email&utm_campaign=mpu_august2025 ) , the GitHub MCP server ( https://github.blog/changelog/2025-07-09-delegate-tasks-to-copilot-coding-agent-from-the-github-mcp-server/?utm_source=github&utm_medium=email&utm_campaign=mpu_august2025 ) , or any base branch ( https://github.blog/changelog/2025-07-23-agents-page-set-the-base-branch-for-github-copilot-coding-agent-tasks/?utm_source=github&utm_medium=email&utm_campaign=mpu_august2025 ) , making it even easier to get Copilot to take on some of the heavy lifting.
* MCP: ( https://github.blog/changelog/2025-07-09-copilot-coding-agent-now-supports-remote-mcp-servers/?utm_source=github&utm_medium=email&utm_campaign=mpu_august2025 ) The agent now has the ability to configure and use MCP servers, either built-in or custom. Connect other applications with Copilot, expanding its context and capabilities.
* New guardrails: ( https://github.blog/changelog/2025-07-15-configure-internet-access-for-copilot-coding-agent/?utm_source=github&utm_medium=email&utm_campaign=mpu_august2025 ) Control the level of access that Copilot has to the internet, and provide tailored guidance for your org with custom instructions ( https://github.blog/changelog/2025-07-23-github-copilot-coding-agent-now-supports-instructions-md-custom-instructions/?utm_source=github&utm_medium=email&utm_campaign=mpu_august2025 ).

( https://github.blog/changelog/2025-07-16-agent-mode-for-jetbrains-eclipse-and-xcode-is-now-generally-available/?utm_source=github&utm_medium=email&utm_campaign=mpu_august2025 )

Powering up IDEs
----------------

* Agent mode added to multiple IDEs: ( https://github.blog/changelog/2025-07-16-agent-mode-for-jetbrains-eclipse-and-xcode-is-now-generally-available/?utm_source=github&utm_medium=email&utm_campaign=mpu_august2025 ) Designed to autonomously perform complex, multi-step tasks, agent mode has become one of the most used features in VS Code. It is now generally available in JetBrains, Xcode, and Eclipse.
* Combining AI and open source: ( https://github.blog/changelog/2025-07-15-github-copilot-in-vs-code-june-release-v1-102/?utm_source=github&utm_medium=email&utm_campaign=mpu_august2025 ) VS Code has open sourced the Copilot Chat extension, allowing bugs to be found and addressed more quickly, and opportunity for community led updates and custom extension solutions.
