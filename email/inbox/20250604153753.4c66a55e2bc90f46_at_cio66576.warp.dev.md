Launch Log 3 is here
********************

MCP is now on GA, and we've overhauled the process for editing agent-generated content. And our Preview build now supports uploading images as context and suggests prompts to save based on your recent Agent interactions

Check out the blog ( https://www.warp.dev/blog/launch-log-3 ) for more details.

( https://youtu.be/ja6dZwWgtd8 )

*****************************
MCP support is fully launched
*****************************

Last Launch Log we shared MCP on Preview— thank you to everyone who provided feedback. Based on that input, we’ve incorporated improvements and are excited to bring MCP to GA.

Add MCP servers to Warp to allow agents to interact with external data sources. We've been using it to debug Sentry errors, pull information from Notion, and connect to Postgres.

 

*****************************
Refine Agent Mode suggestions
*****************************

Refine agent-generated commands. Take another pass through the commands agents create, asking Warp to add feature flags or change commit messages.

 

*******************
New to Warp Preview
*******************

Attach images as prompt context. Upload an image alongside your Prompt, and Warp will include it as context. 

Warp will suggest Prompts for you to save. Prompt chips will show up after your Agent conversations, suggestion reusable prompts.

Thanks for using Warp, and let us know what you think on X ( https://x.com/warpdotdev ), by joining our Community Slack ( https://go.warp.dev/join-preview ), or by filing a GitHub ( https://github.com/warpdotdev/Warp ) issue.
