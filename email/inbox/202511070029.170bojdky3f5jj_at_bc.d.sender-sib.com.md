Greetings, Dify builders!
 

We’ve packed the early‑summer stretch (Jun 1 – Jul 10) with upgrades you can ship today.

**TL;DR:**
==========

- **Two‑way MCP (v1.6.0):** Call any MCP server or publish your own in one click.

- **Enhanced Workflow Debugging (v1.5.0):** Inspect variables, rerun single nodes, fix fast.

- **Azure AI Content Safety Plugin:** On‑prem moderation for text + images.

- **100 K GitHub Stars:** Meet the 17 contributors powering the journey.

**🔌 Built‑in Two‑Way MCP (v1.6.0)**
===================================

Agents and Workflows now speak MCP natively. Add Linear, Notion, Zapier (8 000+ apps) as tools, or expose your own Dify apps as MCP servers for any client to invoke—no glue code required.

 

[Docs](https://b6usd.r.a.d.sendibm1.com/mk/cl/f/sh/7nVU1aA2nfsTSeY62oltNKjy2JTL2Wc/YzMKJHSU79Iw) | [Release Notes](https://b6usd.r.a.d.sendibm1.com/mk/cl/f/sh/7nVU1aA2nfuMSB3P1O0LPh3mCfy7mT8/4KkqdP9SAmPr) | [Blog](https://b6usd.r.a.d.sendibm1.com/mk/cl/f/sh/7nVU1aA2nfwFRhYhzxEnS3NaN2SuWPe/Vht534ouIxoo)

**🐞 Enhanced Workflow Debugging (v1.5.0)**
==========================================

The new variable inspect panel records every node’s last run, so you can replay, edit, and step through complex chains without repeat API calls. Debugging is now evidence‑driven, not guesswork.
 

[Release Notes](https://b6usd.r.a.d.sendibm1.com/mk/cl/f/sh/7nVU1aA2nfy8RE40yWTFUPhOXOxhGMA/uW7w5DmhSp5O) | [Blog](https://b6usd.r.a.d.sendibm1.com/mk/cl/f/sh/7nVU1aA2ng01QkZJx5hhWm1ChlSU0Ig/0SvLDIjHHcsq)
