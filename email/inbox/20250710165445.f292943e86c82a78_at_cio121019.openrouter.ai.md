Hey there,

Thanks for signing up for OpenRouter ( https://openrouter.ai ) – the universal API for accessing, scaling and monitoring the world's best language models from OpenAI, Google, Anthropic, and more – all with a single API key.

Here's how to hit the ground running:

*
🔑 Get your API key

Start making requests right away from your API dashboard ( https://openrouter.ai/keys ).
*
🤖 Try a model in your browser

Explore different models interactively on the Chat playground ( https://openrouter.ai/chat ).
*
📚 Check out our docs

Everything you need to integrate OpenRouter in a few lines of code: Documentation ( https://openrouter.ai/docs ).
*
🧠 Pick your model
Compare models and pricing on the models page ( https://openrouter.ai/models ).
*
💵 Add credits

Top up your account balance to access more model requests: Credits Page ( https://openrouter.ai/credits ).
*
📊 Monitor your activity

Keep track of all of your requests on the Activity Page ( https://openrouter.ai/activity ).

We're excited to see what you build. Got questions? Send an email to support@openrouter.ai or join us in Discord ( https://discord.gg/openrouter ).

Welcome aboard,
