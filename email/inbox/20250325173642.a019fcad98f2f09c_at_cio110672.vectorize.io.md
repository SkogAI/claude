 ✨ What's New

 

Vectorize MCP Server (Beta)

Connect AI assistants like Claude to your data with the Vectorize MCP Server. Securely retrieve documents, extract text, and generate deep research on private data—no custom integrations required.

 

📖 Read the docs ( https://docs.vectorize.io/api/api-mcp-server )

 

Azure AI Search Integration (Beta)

Use Azure AI Search as a vector database in your RAG pipelines. Keep your data in the Microsoft ecosystem, combine keyword and vector search, and scale with Azure’s performance and compliance features.

 

📖 Read the docs ( https://docs.vectorize.io/integrations/vector-databases/azure-ai-search )

 

Supabase Vector Database Integration (Beta)
We’ve added support for Supabase as a vector database in your RAG pipelines. Supabase is a great option for developers getting started with pgVector—it’s simple to configure, open source, and backed by a familiar Postgres foundation.

 

📖 Read the docs ( https://docs.vectorize.io/integrations/vector-databases/supabase )
