Vertical Tabs Reinvented, New iOS Navigation

*****************************
Termius News December Edition
*****************************

Happy holidays! We are wrapping up this year with a bunch of exciting updates. First of all, we reinvented vertical tabs in the form of Workspaces, creating a better way to work with a large number of connections at the same time. We also reimagined navigation in our iOS and iPadOS apps. But first things first.

Ready to see what’s next for Termius on iOS? Then check out our new beta. We’ve made major UI improvements to enhance navigation and streamline your workflows on both iPhone and iPad, including an upgraded SFTP experience.

Try it now via TestFlight: https://testflight.apple.com/join/MnuClabg ( https://testflight.apple.com/join/MnuClabg )

*Workspaces — Reinventing Vertical Tabs*

We introduced horizontal tabs as the main navigation method in mid-2024. While most users embraced this intuitive, modern approach, we discovered it created challenges for those managing large numbers of connections. The narrow top strip proved limiting for power users.

We initially attempted to solve this by enhancing the command palette (CMD+K) with a horizontal list view of open connections. However, since the command palette isn't always visible, this solution was not enough.

Ok, back to the drawing board. We asked ourselves how we could improve the top navigation for the audience with a large number of connections. The first hint came from the other side of the problem. We had this common user request that we could not find the right way to build: the need to easily reopen multiple connections. We realized we already had a foundation for this in our Split View feature, which allows up to 16 simultaneous connections on screen. We just needed a way to reopen them.

Split View needed enhancement, though — users couldn't focus on an individual connection without dragging it out. This was another hint — Focus Mode. We built a quick prototype of Focus Mode for Split View, and it felt like a great answer to all the questions above.

The result is Workspaces — a unified system that makes creating and managing multiple connections more intuitive than ever.

It is quite a big code change that we will be releasing in stages. Our latest release introduces Focus Mode for Split View — simply drag one connection over another and switch modes in any pane. Early next year, we'll launch better workspace management, making it easy to create, save, and reopen workspaces.
