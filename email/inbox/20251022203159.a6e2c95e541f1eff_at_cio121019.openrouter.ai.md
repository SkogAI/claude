Hi there,

We are introducing :exacto - Precision Tool-Calling Endpoints.

These endpoints are focused on higher tool-calling accuracy by routing requests to providers that demonstrate measurably better tool calling performance.

Exacto endpoints ( https://openrouter.ai/models?q=exacto&utm_campaign=Update-22Oct2025&utm_content=Update-22Oct2025&utm_medium=email_action&utm_source=customer.io ) are available for:
- Kimi K2
- DeepSeek v3.1 Terminus
- GLM 4.6 
- GPT‑OSS 120B 
- Qwen3 Coder 

While model weights are identical across providers, real-world inference quality can differs. OpenRouter sees billions of requests per month, giving us a unique view into how models behave across inference stacks and allowing us to curate the most accurate providers for agentic and tool-heavy workloads.

For example - If you are using MCPs, you should try the :exacto variant for better tool calling accuracy. 

Read more about this in the blog post ( https://openrouter.ai/announcements/provider-variance-introducing-exacto?utm_campaign=Update-22Oct2025&utm_content=Update-22Oct2025&utm_medium=email_action&utm_source=customer.io ) or thedocumentation ( https://openrouter.ai/docs/features/exacto-variant?utm_campaign=Update-22Oct2025&utm_content=Update-22Oct2025&utm_medium=email_action&utm_source=customer.io ).

Next up, we have launched two new image models and a stealth model -

* GPT 5 - Image ( https://openrouter.ai/openai/gpt-5-image?utm_campaign=Update-22Oct2025&utm_content=Update-22Oct2025&utm_medium=email_action&utm_source=customer.io ) and GPT 5 - Image mini ( https://openrouter.ai/openai/gpt-5-image-mini?utm_campaign=Update-22Oct2025&utm_content=Update-22Oct2025&utm_medium=email_action&utm_source=customer.io )
* Andromeda Alpha ( https://openrouter.ai/openrouter/andromeda-alpha?utm_campaign=Update-22Oct2025&utm_content=Update-22Oct2025&utm_medium=email_action&utm_source=customer.io ) is the new stealth model. It is a small reasoning model that is optimized for image understanding. All prompts and responses will be logged by the provider to improve their services.
