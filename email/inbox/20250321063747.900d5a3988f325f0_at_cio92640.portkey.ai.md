Here's a quick run down on why you should create virtual keys for SkogAI

They're a way to manage your API access more securely - instead of using keys directly, you use these virtual keys. Think of them like disposable credit cards.

Here's how they help:

* Easy key rotation without changing code
* More control - set budgets & rate limits
* Better tracking - request attribution & usage monitoring

I know it might sound complicated, but it's actually super easy to set up. Look:

GIF showing how to set up virtual keys

Once you've got your virtual key, using it is straightforward. Here's a quick example:

#Python example - more in docs
from portkey_ai import Portkey

portkey = Portkey
( api_key="PORTKEY_API_KEY",
virtual_key="YOUR_NEW_VIRTUAL_KEY_SLUG" )


# Use portkey for all your API requests
response = portkey.chat.completions.create
( model="gpt-3.5-turbo",
messages=[{"role": "user", "content": "Hello!"}] )
