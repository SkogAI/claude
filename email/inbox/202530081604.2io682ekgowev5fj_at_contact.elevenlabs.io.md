ElevenLabs Updates

Hello from the ElevenLabs team,


Here’s a summary of this month’s product releases, platform improvements, and community highlights. For live product updates, follow our [Changelog.](https://elevenlabs.io/docs/changelog)

**Company updates**


[ElevenLabs Agents can now navigate IVR phone trees:](https://elevenlabs.io/blog/ivr-navigation) We’ve added support for generating keypad tones as a system tool across telephony stacks, so ElevenLabs Agents can interact with IVR menus directly.


[ElevenLabs Agents now support Chat Mode:](https://elevenlabs.io/blog/elevenlabs-agents-now-support-chat-mode) We introduced Chat Mode so teams can deploy text‑only agents alongside voice within our Agents Platform via SDK, API, or a single‑line embed.


[Voicemail Detection now available in Agents Platform:](https://elevenlabs.io/blog/voicemail-detection) We released voicemail detection that identifies answering systems in real time and either leaves a custom message or exits, lowering spend on unattended calls. 


[GPT‑5 available in ElevenLabs Agents Platform:](https://elevenlabs.io/blog/gpt-5-available-in-elevenlabs-conversational-ai) We added GPT‑5 as a selectable model to improve reasoning and tool use for complex workflows across UI and API.


[New Trust Center for security and compliance:](https://compliance.elevenlabs.io/) we launched our Trust Center, along with six new certifications, to provide a centralized hub for all our security and compliance documentation.


[Eleven Music is here:](https://elevenlabs.io/blog/eleven-music-is-here) We launched Eleven Music to generate studio‑grade songs from natural‑language prompts with broad commercial‑use support.
