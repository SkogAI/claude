Here’s a roundup for admins of what shipped last month, including a GitHub Universe recap, new ways to interact with and manage AI tools, and some notable tool improvements.

Agent HQ ( https://github.blog/news-insights/company-news/welcome-home-agents/?&utm_source=github&utm_medium=email&utm_campaign=mpu_Nov ) evolves GitHub
--------------------------------------------------------------------------------------------------------------------------------------------------------

* Take charge with the agent control plane: ( https://github.blog/changelog/2025-10-28-enterprise-ai-controls-the-agent-control-plane-are-in-public-preview/?&utm_source=github&utm_medium=email&utm_campaign=mpu_Nov ) Govern your AI agents and audit their use across your enterprise from a single location.
* The mission control consolidates your Copilot coding agent tasks: ( https://github.blog/changelog/2025-10-28-a-mission-control-to-assign-steer-and-track-copilot-coding-agent-tasks/?&utm_source=github&utm_medium=email&utm_campaign=mpu_Nov ) A view for developers to assign Copilot to tasks, steer them as they go, and track their progress.
* Unite any agent into a single platform: ( https://github.blog/news-insights/company-news/welcome-home-agents/?&utm_source=github&utm_medium=email&utm_campaign=mpu_Nov ) Over the coming months, coding agents from Anthropic, OpenAI, Google, Cognition, xAI, and more will become available directly within GitHub as part of a Copilot subscription.

( https://github.blog/changelog/2025-10-28-github-code-quality-in-public-preview/?&utm_source=github&utm_medium=email&utm_campaign=mpu_Nov )

Improving quality across the enterprise
---------------------------------------

* Improve your repositories with GitHub Code Quality: ( https://github.blog/changelog/2025-10-28-github-code-quality-in-public-preview/?&utm_source=github&utm_medium=email&utm_campaign=mpu_Nov ) Scan pull requests and content on your default branch with GitHub Code Quality to find, fix, and report on quality issues.
* We’ve enhanced Copilot code review (CCR): ( https://github.blog/changelog/2025-10-28-new-public-preview-features-in-copilot-code-review-ai-reviews-that-see-the-full-picture/?&utm_source=github&utm_medium=email&utm_campaign=mpu_Nov ) CCR leverages additional tools to provide more dependable content, supports seamless handoff to Copilot coding agent, and is more customizable.
* New tools make managing enterprises easier: ( https://github.blog/changelog/2025-10-23-managing-roles-and-governance-via-enterprise-teams-is-in-public-preview/?&utm_source=github&utm_medium=email&utm_campaign=mpu_Nov ) Take advantage of new roles, management tools, and even custom properties for organizations ( https://github.blog/changelog/2025-10-23-organization-custom-properties-are-now-available-in-public-preview/?&utm_source=github&utm_medium=email&utm_campaign=mpu_Nov ).

( https://github.blog/changelog/2025-10-28-copilot-usage-metrics-dashboard-and-api-in-public-preview/?&utm_source=github&utm_medium=email&utm_campaign=mpu_Nov )

By popular request
------------------

* See who’s using Copilot and how much: ( https://github.blog/changelog/2025-10-28-copilot-usage-metrics-dashboard-and-api-in-public-preview/?&utm_source=github&utm_medium=email&utm_campaign=mpu_Nov ) Track Copilot usage across your enterprise, available as a dashboard or through a corresponding API.
* Give Copilot access to internal resources: ( https://github.blog/changelog/2025-10-28-copilot-coding-agent-now-supports-self-hosted-runners/?&utm_source=github&utm_medium=email&utm_campaign=mpu_Nov ) Run Copilot coding agent on your own GitHub Actions runners to give it access to resources not available on the internet.
* Create immutable releases: ( https://github.blog/changelog/2025-10-28-immutable-releases-are-now-generally-available/?&utm_source=github&utm_medium=email&utm_campaign=mpu_Nov ) Instill trust in your product by creating releases that can’t be modified.
* Specialize your Copilot experience with custom agents ( https://github.blog/changelog/2025-10-28-custom-agents-for-github-copilot/?&utm_source=github&utm_medium=email&utm_campaign=mpu_Nov ) : Create your own custom agent to use or take advantage of one created by one of our trusted partners.

( https://github.blog/changelog/2025-10-16-copilot-coding-agent-can-now-search-the-web/?&utm_source=github&utm_medium=email&utm_campaign=mpu_Nov )

Copilot coding agent levels up… again
-------------------------------------

* *Access Copilot from more places:* Call in Copilot to help you directly from Linear ( https://github.blog/changelog/2025-10-28-github-copilot-for-linear-available-in-public-preview/?&utm_source=github&utm_medium=email&utm_campaign=mpu_Nov ) or Slack ( https://github.blog/changelog/2025-10-28-work-with-copilot-coding-agent-in-slack/?&utm_source=github&utm_medium=email&utm_campaign=mpu_Nov ).
* Copilot-generated code is automatically analyzed: ( https://github.blog/changelog/2025-10-28-copilot-coding-agent-now-automatically-validates-code-security-and-quality/?&utm_source=github&utm_medium=email&utm_campaign=mpu_Nov ) All code produced by Copilot is now filtered through GitHub’s security and quality validation tools.
* Copilot coding agent gets more resources: ( https://github.blog/changelog/2025-10-16-copilot-coding-agent-can-now-search-the-web/?&utm_source=github&utm_medium=email&utm_campaign=mpu_Nov ) When you use Copilot coding agent, it now can search the web for additional context and resources.
