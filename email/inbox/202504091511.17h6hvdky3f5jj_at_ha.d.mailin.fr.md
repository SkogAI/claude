Summer's wrapping up, and we've shipped features that'll make your workflows smoother and faster. Here's what landed in v1.7.0 through v1.8.0.

==========

- **OAuth authentication for tool plugins (e.g., Gmail, GitHub…)**

🔐 OAuth Support for Tools (v1.7.0)
==================================

Tool plugins now support OAuth 2.0. Instead of pasting API keys, users click to authorize and maintain long lived sessions with refresh tokens. This keeps credentials secure and simplifies multi-account workflows.

 

If you are self-hosting, register OAuth apps with the services you use and add the client credentials to your Dify setup. Cloud users already have OAuth preconfigured for the most popular services.

**🔍 Visual Workflow Navigation (v1.7.2)**
=========================================

Hold **Shift** and click a node to spotlight its connections while the rest fade out. A new relationship panel gives you a clear view of data dependencies. With **Cmd/Ctrl+K**, you can instantly search and jump to any app, knowledge base, plugin, or workflow node. Version 1.8.0 makes it even smoother with slash commands for fast settings changes.

 

[Release Notes](https://b6usd.r.a.d.sendibm1.com/mk/cl/f/sh/7nVU1aA2nfwFRhYhzxEnS3NaN2SuWPe/Jo6B0bolALDk)

**✨ Smart Prompt & Code Fixes (v1.8.0)**
========================================

When prompts aren't working as expected, the optimizer analyzes your last run and ideal output to suggest improvements. When code throws errors, autofix generates corrections based on the error trace. Every optimization saves as a new version, compare and rollback anytime.

**Start Node with Defaults**
----------------------------

Your workflows now run even without input. Add default values to Start node fields to power scheduled reports, demo workflows, or to keep things stable when upstream data is missing. Use URL parameters to set up variations for each channel in seconds.

**Accurate Agent Token Tracking**
---------------------------------

Agent nodes now split input and output tokens correctly, instead of counting everything as output. This gives you precise usage data across downstream nodes, enabling cost controls and even automatic model switching when thresholds are reached.

**Multi-Credential Management**
-------------------------------

Now you can add multiple credentials under the same model provider and decide which key each model should use. With load balancing feature, requests are distributed across keys, and if one hits a limit or fails, the system instantly switches to a backup to keep your service safe and reliable.

**Faster Workflows**
--------------------

The new async execution engine turns parallel branches into true concurrency. Database writes no longer block progress, so workflows that previously required the sum of all branch execution time now finish in about the time of the longest branch.
