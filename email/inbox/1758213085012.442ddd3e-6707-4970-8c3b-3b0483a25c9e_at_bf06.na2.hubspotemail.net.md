Learn to build agents for complex, long-running tasks

Many agents today follow the same simple pattern: run in a loop, call tools. That architecture works well enough, but it breaks down as tasks get more complex. These agents often struggle to plan over longer horizons or manage more demanding workflows.

Major model providers have built “deep” agents in domains like coding and research. These agents dive deeper – they’re able to plan complex tasks and carry them out over longer time horizons.

Today, companies of all sizes – from startups to large enterprises – are building their own Deep Agents.

There are four key features that set Deep Agents apart from regular agents:

- Planning – keeps agents on track
- File system – allows agents to offload context
- Sub-agents – act as focused specialists
- Prompting – provides agents with detailed instructions

Our latest LangChain Academy course, Deep Agents with LangGraph, shows you how to combine these pieces with LangGraph to orchestrate long-running, multi-agent workflows.

By the end, you’ll have the skills to design, implement, and deploy your own Deep Agent – ready to tackle complex, long-running tasks in your domain.
