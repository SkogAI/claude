How Hugging Face Is Using E2B to Replicate DeepSeek-R1

 
The Open R1 project by Hugging Face reverse-engineered the missing pieces of DeepSeek’s data and training pipeline.
The problem was that executing LLM-generated code locally poses many risks; the model may accidentally corrupt your hard drive or
even rm -rf your entire home directory. Read about how Hugging Face overcame this with E2B sandboxes for code execution.
[https://e2b.dev/blog/how-hugging-face-is-using-e2b-to-replicate-deepseek-r1]

Built with E2B

 


Manus

Manus [https://manus.im/] is a deep research AI agent with an ability to conduct a complex research and create reports with custom
data visualizations powered by E2B runtime.
 


Surf

Surf is an open source playground for running OpenAI's new Computer Use Agent with E2B Desktop SDK. The whole app is open source
[https://github.com/e2b-dev/surf].
 


Rork app

You can build fast mobile apps. Rork [https://rork.app/] is using E2B as the environment for running the script for generating the
apps.
 


Hugging Face Open-R1

Hugging Face uses E2B code execution to fully replace DeepSeek-R1. E2B is implemented as part of the reward function for code
execution in the Open-r1 project [https://github.com/huggingface/open-r1].
 


Deep Research in Perplexity

E2B is powering the code execution, data analysis & producing charts in the new Perplexity Deep Research
[https://www.perplexity.ai/hub/blog/introducing-perplexity-deep-research] agent.
 


Factory AI

Factory released a platform for automating code-related processes with AI agents. It's now possible to use E2B-powered workflows
[https://app.factory.ai/workflows/library/vAYGV2YYMF48FsDxv3IQ], for example, for building desktop AI apps.
 


Endex using E2B in their Financial Analyst

Endex [https://openai.com/index/endex/] is building the future of financial analysis powered by OpenAI’s reasoning models. They
run the scripts for data analysis inside the E2B Sandbox environment.
 


Cursor running code with E2B MCP server

You can install the E2B MCP server from GitHub [https://github.com/e2b-dev/mcp-server] and then add it to Cursor via its settings
to get secure and isolated cloud runtime.
