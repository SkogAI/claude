It's Dane from Plandex [https://plandex.ai]—I wanted to give you a quick update. It's been a busy couple of weeks since the v2
release, and quite a few bugs have been fixed across a handful of releases. If you tried the v2 right after launch and ran into
frequent model errors or other issues, those are most likely fixed now.

Apart from the fixes, which have been the top priority, you can now use Plandex v2's debugging features with browser apps.
Previously, debugging required terminal output to work well, which excluded frontend/browser apps. Now Plandex can launch a
browser and redirect errors and console logs to the terminal for debugging. This requires Chrome to be installed, but otherwise
it's built-in and enabled by default. Here's a quick demonstration:



https://www.youtube.com/watch?v=g-_76U_nK0Y&ts



Last but not least, there are some new models and model packs available:
 
 * Claude Sonnet 3.7 thinking is now available as a built-in model. Try the reasoning model pack for more challenging tasks.
   

 * Gemini 2.5 pro (free/experimental version) is now available. Try the gemini-exp model pack to use it alongside the default
   models for the best results.
   

 * DeepSeek V3 03-24 version is available as a built-in model and is now used in the oss pack in the in the the coder role.
   

 * OpenAI GPT 4.5 is available as a built-in model. It's not in any model packs so far due to rate limits and high cost, but is
   available to use via set-model
