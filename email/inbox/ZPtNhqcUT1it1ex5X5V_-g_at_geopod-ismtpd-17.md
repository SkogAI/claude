The latest launches from GitHub. See what’s new and how to get started.

It’s been a busy month here at GitHub as we’ve worked to bring several products and features out of preview and into general availability. The agentic era is here, and there are plenty of ways you can tap into the power of AI today.

( https://github.blog/changelog/2025-04-04-announcing-github-copilot-pro/?utm_source=github&utm_medium=email&utm_campaign=mpu_april )

Right-size your GitHub Copilot and security subscriptions
---------------------------------------------------------

We're providing you with more flexible service plans so you can get the features you need and only pay for what you use.

* Copilot Pro+ ( https://github.blog/changelog/2025-04-04-announcing-github-copilot-pro/?utm_source=github&utm_medium=email&utm_campaign=mpu_april ) : Introducing a new service tier, Copilot Pro+, offering 1,500 premium requests compared to 300 for Pro. Both Pro+ and Enterprise offer unlimited access to the base model.
* GitHub Secret Protection and GitHub Code Scanning ( https://github.blog/changelog/2025-04-01-github-secret-protection-and-github-code-security-for-github-enterprise?utm_source=github&utm_medium=email&utm_campaign=mpu_april ) : We now offer GitHub Advanced Security standalone products.
* Plus more Copilot Chat models ( https://github.blog/changelog/2025-04-04-multiple-new-models-are-now-generally-available-in-github-copilot/?utm_source=github&utm_medium=email&utm_campaign=mpu_april ) : For any paid Copilot plan, the following models are now GA: Anthropic Claude 3.5 Sonnet and Claude 3.7 Sonnet, Google Gemini 2.0, and OpenAI o3-mini, enabling you to tackle coding tasks one minute—then dive into deeper reasoning the next.

( https://github.blog/changelog/2025-04-04-copilot-code-review-now-generally-available/?utm_source=github&utm_medium=email&utm_campaign=mpu_april )

GitHub Copilot evolves from an AI coding assistant to an AI collaborator
------------------------------------------------------------------------

Our first agentic AI features are now generally available:

* GitHub Copilot code review ( https://github.blog/changelog/2025-04-04-copilot-code-review-now-generally-available/?utm_source=github&utm_medium=email&utm_campaign=mpu_april ) : Offload basic reviews to a copilot with this agent that finds bugs, potential performance problems, and even suggests automatic fixes.
* GitHub Copilot agent mode ( https://github.blog/changelog/2025-04-03-github-copilot-in-vs-code-march-release-v1-99/?utm_source=github&utm_medium=email&utm_campaign=mpu_april ) : Work with a pair programming agent directly in VS Code. It can gather context, iterate on outputs, find and fix errors, and edit multiple files.
* MCP support for VS Code ( https://github.blog/changelog/2025-04-04-github-mcp-server-public-preview/?utm_source=github&utm_medium=email&utm_campaign=mpu_april ) : To help empower your agents, we’re previewing support in VS Code for Model Context Protocol (MCP), a standard for connecting AI models with data sources and tools like APIs and databases.

( https://github.blog/changelog/2025-04-04-github-copilot-in-vs-code-march-release-v1-99/#ux-improvements-help-you-work-faster-and-stay-focused/?utm_source=github&utm_medium=email&utm_campaign=mpu_april )

But wait, there's even more now generally available in Copilot
--------------------------------------------------------------

* Next edit suggestions (NES) ( https://github.blog/changelog/2025-04-04-github-copilot-in-vs-code-march-release-v1-99/#ux-improvements-help-you-work-faster-and-stay-focused/?utm_source=github&utm_medium=email&utm_campaign=mpu_april ) : A refreshed experience for quicker code completion within existing code.
* GPT-4o Copilot code completion model ( https://github.blog/changelog/2025-03-27-gpt-4o-copilot-your-new-code-completion-model-is-now-generally-available/?utm_source=github&utm_medium=email&utm_campaign=mpu_april ) : A faster, better default model for code complete, based on our GitHub trained version of GPT-4o.

( https://github.blog/changelog/2025-03-24-enterprise-custom-properties-enterprise-rulesets-and-pull-request-merge-method-rule-are-all-now-generally-available/?utm_source=github&utm_medium=email&utm_campaign=mpu_april )

Standardize and scale repos across your enterprise
--------------------------------------------------

* Enterprise rulesets ( https://github.blog/changelog/2025-03-24-enterprise-custom-properties-enterprise-rulesets-and-pull-request-merge-method-rule-are-all-now-generally-available/?utm_source=github&utm_medium=email&utm_campaign=mpu_april ) : Manage branch rules and push protection through rulesets that target multiple repositories in your enterprise.
* Enterprise custom properties ( https://github.blog/changelog/2025-03-24-enterprise-custom-properties-enterprise-rulesets-and-pull-request-merge-method-rule-are-all-now-generally-available/?utm_source=github&utm_medium=email&utm_campaign=mpu_april ) : More flexibility to define decorations for your repos, like compliance framework expectations and severity levels for data sensitivity. Rulesets can then be applied to these custom properties, at scale.
