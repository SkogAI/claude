###[**Community Spotlight: MCP Server for NetBird**](https://az5n7.r.a.d.sendibm1.com/mk/cl/f/sh/1t6Af4OiGsE8LIklyUF7NVOEC4dLWQ/6bUvZddw6Rkx)


Community contributor **aantti** has developed an **MCP (Model Context Protocol) server for NetBird**, enabling seamless integration of NetBird's network data into LLM-powered tools like Claude Desktop and Codeium Windsurf.

 

This tool provides read-only access to NetBird resources such as peers, groups, policies, and networks, facilitating enhanced observability and automation. It's particularly beneficial for users aiming to incorporate NetBird insights into their AI-driven workflows.

 

[Explore the project here.](https://az5n7.r.a.d.sendibm1.com/mk/cl/f/sh/1t6Af4OiGsEag0xCMqL2XlS3Ztc5EU/N9Yu8wR_yEUK)

 


###New in NetBird  - Auditor and Network Admin Roles
 


**Auditor**: Can view all configurations, but cannot make any changes.

 

**Network Admin**: Can manage network configurations - including access policies, DNS settings, networks, and routes. However, they can only view (not modify) user and device details, as well as general account settings.

 

**New Invoices Section (Cloud Only)**
A new **"Invoices"** section is now available under **Settings** for cloud users. Account Owners and Billing Admins can view and export all available invoices.

 

We'll [update the documentation](https://az5n7.r.a.d.sendibm1.com/mk/cl/f/sh/1t6Af4OiGsF30j9clCQxi1VsxiaowY/WTGVL-4O2EIR "NetBird Docs") soon.
