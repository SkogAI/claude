More and more of the traffic on the web is driven not by humans, but by agents — over the month of January alone, the number of weekly requests generated by AI agents more than doubled across the Cloudflare network. This is unsurprising, given the rise of accessible agents like OpenClaw that overnight gave everyone who wanted one a personal assistant (and by the way, if you haven’t set one up yet, you can do so very quickly on Cloudflare↗ — no Mac mini required!). If you’re building agents, we've shipped several improvements to the Agents SDK↗ we recommend checking out and platform updates that support better development on agents like our Workflows visualizer, extended subrequests limits, and more! We didn't stop there. As a site owner, you can now make your site agent-ready with the new Markdown for Agents↗ feature, which will automatically serve markdown content when agents send the appropriate "Accept: text/markdown header" This is just the tip of the iceberg of everything that’s new on our platform. For the full download, hope you keep reading! Self-host OpenClaw Forget the Mac mini. If you want to run OpenClaw (the open-source AI agent & personal assistant) without dedicated hardware, we got you. Meet Moltworker, a middleware Worker and adapted scripts that lets you run Moltbot on Cloudflare's Sandbox SDK and Developer Platform APIs.
