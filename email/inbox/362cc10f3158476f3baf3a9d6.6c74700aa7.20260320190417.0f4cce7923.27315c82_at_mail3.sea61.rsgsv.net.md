** Agent Mode Gets Better
------------------------------------------------------------

** SSH Remotes
------------------------------------------------------------

You can now push and pull Dolt databases using SSH (https://dolthub.us5.list-manage.com/track/click?u=362cc10f3158476f3baf3a9d6&id=0baeb99bcb&e=6c74700aa7) . This was a long standing request from folks who already have SSH infrastructure in place. If you're already using SSH-based Git workflows, you'll feel right at home with Dolt SSH remotes.


** lazydolt
------------------------------------------------------------

If you've used lazygit (https://dolthub.us5.list-manage.com/track/click?u=362cc10f3158476f3baf3a9d6&id=bf485dfff8&e=6c74700aa7) , you'll love lazydolt (https://dolthub.us5.list-manage.com/track/click?u=362cc10f3158476f3baf3a9d6&id=730313f6f7&e=6c74700aa7) . It's a Terminal User Interface built in Go that lets you navigate branches, view diffs, manage commits, and run database operations without leaving your terminal. The interesting bit is that it was built by coding agents. Things that were not worth building a year ago are suddenly worth building.
