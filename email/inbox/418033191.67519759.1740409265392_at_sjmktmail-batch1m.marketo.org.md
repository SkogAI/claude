Dictionaries are in-memory key-value pairs that can be used with your queries.

If you’re familiar with Redis or with JSON objects, using dictionaries should seem familiar to you. Here is an example of a SELECT query retrieving data from a dictionary:

SELECT dictGet(my_dictionary, my_column, ‘my_id’);

In this example, ClickHouse queries my_dictionary and returns the column my_column from the entry with the id that matches my_id. The equivalent in JavaScript would be:

myObj[‘my_id’].myColumn;

Dictionaries are especially useful for lookup tables used to make JOINs more performant.
