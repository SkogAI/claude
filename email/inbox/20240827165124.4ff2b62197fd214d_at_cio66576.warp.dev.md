Introducing unlimited AI for teams

+ more AI requests for every dev on Free and Pro

With Agent Mode in Warp ( https://x.com/warpdotdev/status/1828462906519834627 ), you can describe any task you’d like to complete and let AI walk you through the task from start to finish. Start by typing a phrase on the command line in natural language. You can give directions like:
* “Test database connectivity from my kubernetes cluster”
* “Set up a new python project for me,” or
*
“Create a new git branch remotely”

You can collaborate with AI to debug errors, teach you CLIs, or even build new product features with you.

These multi-step workflows require more AI requests to complete. That’s why we’re raising the limits for AI on every Warp plan today.

Learn more ( https://www.warp.dev/blog/assign-unlimited-tasks-to-ai-on-warps-team-plan )
