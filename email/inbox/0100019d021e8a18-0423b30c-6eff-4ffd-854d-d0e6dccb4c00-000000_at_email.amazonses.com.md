Hello Emil,

We're sharing some major updates to Letta and Letta Code!


AUTO MODE

We recently introduced letta/auto and letta/auto-fast model handles, which route to models optimized for Letta Code. We recommend using these for the best cost and performance.


NEW PERSONAL PLANS

We're introducing Personal Plans designed for individual (non-automated) use - ideal if you're using Letta hands-on via Letta Code https://github.com/letta-ai/letta-code or the chat interface. Personal Plans provide usage quotas across various providers and Letta's auto endpoints.

You can upgrade to the new plans on the usage page https://app.letta.com/settings/organization/usage.

https://www.letta.com/pricing




If you're using Letta to build applications or run automated workloads, the API Plan https://www.letta.com/pricing is the right fit. If you were on the legacy Pro Plan, you've been automatically migrated to the API Plan.


LETTA'S NEXT PHASE

We're shifting our product focus toward computer-use agents like Letta Code. As part of this transition, we'll be sunsetting some existing features. We know changes like this can be disruptive, so please see our announcement https://www.letta.com/blog/our-next-phase for full details on what's changing and the timeline.

Let us know if you have any questions or concerns (you can rely to this email)!
