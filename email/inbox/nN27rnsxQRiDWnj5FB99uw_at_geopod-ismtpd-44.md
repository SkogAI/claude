Get more info, right where you work
-----------------------------------

* Get Copilot code reviews right in the CLI: ( https://github.blog/changelog/2026-03-11-request-copilot-code-review-from-github-cli/?utm_source=github&utm_medium=email&utm_campaign=FY26APR-WW-LCM-PL-X-AA-Admin-PRM-X-701Kg000000PiEPIA0 ) Have GitHub Copilot start a code review directly from the Copilot CLI without switching to a browser.
* Track more details about Copilot cloud agent (CCA) tasks: ( https://github.blog/changelog/2026-03-19-more-visibility-into-copilot-coding-agent-sessions/?utm_source=github&utm_medium=email&utm_campaign=FY26APR-WW-LCM-PL-X-AA-Admin-PRM-X-701Kg000000PiEPIA0 ) The session logs for CCA contain more information, particularly around startup and delegating tasks to subagents.
* Action run summaries encompass more complete info: ( https://github.blog/changelog/2026-03-26-view-agentic-workflow-configs-in-the-actions-run-summary/?utm_source=github&utm_medium=email&utm_campaign=FY26APR-WW-LCM-PL-X-AA-Admin-PRM-X-701Kg000000PiEPIA0 ) View the Markdown configuration of agentic workflows directly in the run summary.
* Copilot usage metrics show what model was used for auto: ( https://github.blog/changelog/2026-03-20-copilot-usage-metrics-now-resolve-auto-model-selection-to-actual-models/?utm_source=github&utm_medium=email&utm_campaign=FY26APR-WW-LCM-PL-X-AA-Admin-PRM-X-701Kg000000PiEPIA0 ) Identify exactly which models are being used across your organization, even when auto model selection is used.
* Track Copilot cloud agent usage: ( https://github.blog/changelog/2026-03-25-copilot-usage-metrics-now-identify-active-copilot-coding-agent-users/?utm_source=github&utm_medium=email&utm_campaign=FY26APR-WW-LCM-PL-X-AA-Admin-PRM-X-701Kg000000PiEPIA0 ) Copilot usage metrics now distinguish between direct model usage and CCA model usage.

( https://github.blog/changelog/2026-03-19-copilot-coding-agent-now-starts-work-50-faster/?utm_source=github&utm_medium=email&utm_campaign=FY26APR-WW-LCM-PL-X-AA-Admin-PRM-X-701Kg000000PiEPIA0 )

Making Copilot faster and more efficient
----------------------------------------

* Copilot cloud agent starts working 50% faster: ( https://github.blog/changelog/2026-03-19-copilot-coding-agent-now-starts-work-50-faster/?utm_source=github&utm_medium=email&utm_campaign=FY26APR-WW-LCM-PL-X-AA-Admin-PRM-X-701Kg000000PiEPIA0 ) Take advantage of the shorter startup times whenever you hand off work to Copilot.
* Copilot cloud agent completes task up to 2% faster: ( https://github.blog/changelog/2026-03-17-copilot-coding-agent-works-faster-with-semantic-code-search/?utm_source=github&utm_medium=email&utm_campaign=FY26APR-WW-LCM-PL-X-AA-Admin-PRM-X-701Kg000000PiEPIA0 ) CCA uses the semantic search capabilities to find necessary code faster, giving a speed improvement with no change in quality.
* Copilot can make changes to pull requests: ( https://github.blog/changelog/2026-03-24-ask-copilot-to-make-changes-to-any-pull-request/?utm_source=github&utm_medium=email&utm_campaign=FY26APR-WW-LCM-PL-X-AA-Admin-PRM-X-701Kg000000PiEPIA0 ) Mention @copilot in a comment on a pull request to have a CCA work directly on that pull request.
* Specify what model to use in pull request comments: ( https://github.blog/changelog/2026-03-05-pick-a-model-for-copilot-in-pull-request-comments/?utm_source=github&utm_medium=email&utm_campaign=FY26APR-WW-LCM-PL-X-AA-Admin-PRM-X-701Kg000000PiEPIA0 ) Use the pulldown to specify which model Copilot should use when addressing your comment in a pull request.

( https://github.blog/changelog/2026-03-18-gpt-5-3-codex-long-term-support-in-github-copilot/?utm_source=github&utm_medium=email&utm_campaign=FY26APR-WW-LCM-PL-X-AA-Admin-PRM-X-701Kg000000PiEPIA0 )

Solidifying your organization
-----------------------------

* Long-term support added for GPT-5.3-Codex: ( https://github.blog/changelog/2026-03-18-gpt-5-3-codex-long-term-support-in-github-copilot/?utm_source=github&utm_medium=email&utm_campaign=FY26APR-WW-LCM-PL-X-AA-Admin-PRM-X-701Kg000000PiEPIA0 ) Count on model stability by leveraging long-term-support (LTS) models. GPT-5.3-Codex is the first LTS model and will remain available into 2027 for Business and Enterprise users.
