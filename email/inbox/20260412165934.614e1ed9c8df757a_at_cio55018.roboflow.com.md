Not sure where to start?
------------------------

1. Want a
model in < 5 minutes?
Try Rapid, our tool to build an object detection model in minutes ( https://rapid.roboflow.com/?utm_campaign=Onboarding+Campaign+%28Gust+-+02%2F2026%29&utm_content=Welcome+to+Roboflow+%28app%29+-+Max&utm_medium=email_action&utm_source=email ).

2. Already have images?
Upload & label them. AI-assisted tools ( https://roboflow.com/annotate?utm_campaign=Onboarding+Campaign+%28Gust+-+02%2F2026%29&utm_content=Welcome+to+Roboflow+%28app%29+-+Max&utm_medium=email_action&utm_source=email ) make labeling 10× faster.

3. Trying an existing model?

4. Exploring ideas?

Use a template. Jump into sample use cases ( https://templates.roboflow.com/?utm_campaign=Onboarding+Campaign+%28Gust+-+02%2F2026%29&utm_content=Welcome+to+Roboflow+%28app%29+-+Max&utm_medium=email_action&utm_source=email ) to see how everything works.

5) Deployment infrastructure
Run models directly on Roboflow's infrastructure ( https://roboflow.com/deploy?utm_campaign=Onboarding+Campaign+%28Gust+-+02%2F2026%29&utm_content=Welcome+to+Roboflow+%28app%29+-+Max&utm_medium=email_action&utm_source=email ) through an API, self-host deployment using our open source inference server, or use native SDKs.
