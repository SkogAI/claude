Here's a roundup of what shipped in the last month, including new tools to track and control your AI spending, plus enhanced code review capabilities.

( https://github.blog/changelog/2025-06-24-github-models-now-supports-moving-beyond-free-limits/?utm_source=github&utm_medium=email&utm_campaign=mpu_july2025 )

More plans, more control
------------------------

* GitHub Models now supports bring your own key (BYOK) ( https://github.blog/changelog/2025-06-24-github-models-now-supports-moving-beyond-free-limits/?utm_source=github&utm_medium=email&utm_campaign=mpu_july2025 ) : GitHub Models now enables you to use your own keys and supports pay-as-you-go billing to help you scale beyond the free tier.
* Granular budget management ( https://github.blog/changelog/2025-05-09-sku-level-budgets-now-available-for-enhanced-cost-control?utm_source=github&utm_medium=email&utm_campaign=mpu_july2025 ) : Administrators can now set precise spending limits per SKU for all metered GitHub products, including premium GitHub Copilot requests ( https://github.blog/changelog/2025-06-18-update-to-github-copilot-consumptive-billing-experience/?utm_source=github&utm_medium=email&utm_campaign=mpu_july2025 ).
* Coding agent expands to Copilot Business users ( https://github.blog/changelog/2025-06-24-github-copilot-coding-agent-is-now-available-for-copilot-business-users/?utm_source=github&utm_medium=email&utm_campaign=mpu_july2025 ) : Copilot Business subscribers can now delegate tasks to Copilot to work in the background. You can enable the feature through the "Copilot coding agent" policy.
* Summarized billing usage reports ( https://github.blog/changelog/2025-06-19-summarized-billing-usage-reports-now-available?utm_source=github&utm_medium=email&utm_campaign=mpu_july2025 ) : Generate summarized billing reports for up to one year, or detailed reports (including usernames and workflow paths) for up to 31 days.

( https://github.blog/changelog/2025-06-12-remote-github-mcp-server-is-now-available-in-public-preview/?utm_source=github&utm_medium=email&utm_campaign=mpu_july2025 )

MCP is now easier and more powerful
-----------------------------------

* The remote GitHub MCP Server ( https://github.blog/changelog/2025-06-12-remote-github-mcp-server-is-now-available-in-public-preview/?utm_source=github&utm_medium=email&utm_campaign=mpu_july2025 ) provides easy setup, simple authentication, and automatic updates. Integrate the remote GitHub MCP server with GitHub Copilot in VS Code, Claude Code, and more.
* New MCP features in VS Code ( https://github.blog/changelog/2025-06-13-github-copilot-in-vs-code-may-release-v1-101/?utm_source=github&utm_medium=email&utm_campaign=mpu_july2025 ) : Reusable prompts and resources accessible via slash commands and context attachments, enable smarter workflows directly within your editor.

( https://github.blog/changelog/2025-06-13-copilot-code-review-customization-for-all/?utm_source=github&utm_medium=email&utm_campaign=mpu_july2025 )

Code review gets smarter
------------------------

* Set custom instructions for GitHub Copilot code review ( https://github.blog/changelog/2025-06-13-copilot-code-review-customization-for-all/?utm_source=github&utm_medium=email&utm_campaign=mpu_july2025 ) : Copilot can review your code with tailored instructions for team standards and preferences, ensuring consistent and relevant feedback across all pull requests.
* Scheduled reminders for pull requests ( https://github.blog/changelog/2025-06-11-gain-more-control-and-clarity-with-scheduled-reminders-for-pull-requests?utm_source=github&utm_medium=email&utm_campaign=mpu_july2025 ) : Enhanced scheduled reminders for Slack and Microsoft Teams now give you more control with reminders for up to 20 pull requests from up to five repositories, with transparent repository selection to help you focus on what matters most.

* Configuring which secret scanning patterns are included in push protection is in public preview ( https://github.blog/changelog/2025-06-25-configuring-which-secret-scanning-patterns-are-included-in-push-protection-is-in-public-preview/?utm_source=github&utm_medium=email&utm_campaign=mpu_july2025 )
