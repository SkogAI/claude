Sunsetting Roo Code and Launching Roomote

 
Hi there,
 
We wanted to let you know directly that we're sunsetting the Roo Code suite of tools on May 15th. This includes the Roo Code VS
Code Extension, Roo Code Cloud, and Roo Code Router.

First, thank you. Your trust and contributions helped us cross 3M extension downloads and push real innovation in this space.
We're grateful.

That said, we don't believe IDEs are the future of coding. To keep pushing the future forward, we had to make this hard call. You
can read more about why here [https://x.com/mattrubens/status/2046636598859559114].
 
What this means for you:
 * We'll continue to support all existing Roo Code products through May 15th.

 * On May 15th, we'll shut down Roo Code Cloud and Roo Code Router, refund any unused balances, and archive the Roo Code Extension
   repo.

 * For billing questions after that, you can reach out to billing@roocode.com.

 * You can keep up with any changes/updates via our sunset blog post
   [https://roocode.com/blog/sunsetting-roo-code-extension-cloud-and-router] or Discord.

 
Looking for an alternative?
 * If you want a model-agnostic open-source extension, we recommend Cline [https://cline.bot/]. They've incorporated much of what
   we built and are excited to welcome Roo Code users.

 
If you're a fan of our cloud agents or curious about what we're building next, check out our new home at roomote.dev
[https://roomote.dev/].
 
Thanks for being part of the journey,
The Roo Code Team
