Hello,

Today, we *dropped the price of* *OpenAI o3* ( https://platform.openai.com/docs/models/o3 ) *by 80%* , bringing the cost down to $2 / 1M input tokens and $8 / 1M output tokens. We optimized our inference stack that serves o3—this is the same exact model, just cheaper.

Based on evals ( https://openai.com/index/introducing-o3-and-o4-mini/ ) and customer feedback, we recommend you try o3 for coding (this model now costs the same per token as GPT-4.1 and is cheaper than GPT-4o!) and for any tasks that require agentic tool calling, function calling, and instruction following.

*We also introduced OpenAI* *o3-pro* ( https://platform.openai.com/docs/models/o3-pro ) *in the API* —a version of o3 that uses more compute to think harder and provide reliable answers to challenging problems. We know that o1-pro was often too expensive to use in production, so we’re excited to be able to lower the price by 87% with o3-pro—offering the model at $20 / 1M input tokens and $80 / 1M output tokens. o3-pro is available as ‘o3-pro-2025-06-10’ in the Responses API and supports image inputs, function calling, and Structured Outputs. Since o3-pro is designed to tackle tough problems, some requests may take several minutes to finish. To avoid timeouts, try using the new background mode ( https://platform.openai.com/docs/guides/background ) in the Responses API.

Learn more about o3 ( https://platform.openai.com/docs/models/o3 ) and o3-pro ( https://platform.openai.com/docs/models/o3-pro ) , and start testing o3-pro ( https://platform.openai.com/playground/prompts?models=o3-pro ) in the Playground.
