Introducing a Reimagined Visual History

GitLens 17.1 is now available with a completely rebuilt Visual History feature. This major upgrade transforms the former Visual File History into a full repository exploration system, offering seamless navigation from file → folder → repo levels via interactive breadcrumbs.

What's new in Visual History?

🔍 Zoom Out to See the Big Picture

Navigate seamlessly from file to folder to repository level using the new interactive breadcrumb interface. Start with a single file and expand your view to understand changes in context across your entire codebase.

Discover Changes Across Branches

With the new "View All Branches" mode, slice your visualization by branch instead of author to immediately identify:

- Changes that haven't been merged into your main branch
- Work in progress on feature branches
- Code evolution across multiple development streams

🏠 One-Click Access from Home

Launch Visual History directly from the Home view with a prominent new button right next to Graph, making repository visualization just one click away.
