Hey Streamlitters, 
 
Get excited. Streamlit 1.42 is here, bringing built-in authentication (yes, really!) and powerful dataframe updates giving you more control and flexibility. Keep reading to learn more. Plus, see winning hackathon apps and cool creations from the community.
&#128680;  Release 1.42
Highlights 



	- &#x1F469;&#x200D;&#x1F4BB; Authentication: Introducing st.login() and st.logout() to easily authenticate users with any OpenID Connect provider.
	- &#128202; Dataframe updates:



	- Drag and drop columns to rearrange them.
	- Use column menus for sorting and pinning.
	- Support for categorical indices.
	- See a hover highlight on rows.
	- Tooltips show cell value type inconsistencies.



	- &#10083;&#65039; Markdown Support: Now available in st.table and st.image captions.
