Video of the Week

This week’s video is a quick overview of the platform and the basic workflow you can use to test and improve your prompts.
 

https://youtu.be/CCjD26en_Tc

 
TL;DW
 * Write or paste your prompt in the Prompt Manager

 * Run it a few times in Preview to see how it performs

 * Create a dataset and an evaluation to test it

 * Run an experiment to compare versions or models

 * Publish your improved prompt and track its performance

 


This Week’s Highlights ✨

Some of the features we've launched include:
 
 * New Chat UI: A smoother, faster interface to build and test your prompts. You’ll see clearer reasoning steps and cleaner
   interactions.

 



 
 
 * Improved Simulations: Experiments now behave more like real runs. Every tool call is simulated, so your results better reflect
   production.

 



 
 
 * Multimodal Output Support: Your agents can now generate images, files, and videos directly from Latitude (try it with Google
   Gemini’s Nano Banana model 🍌).

 



 
 
 * Composite Scores: Measure your prompts’ overall quality in one metric that combines accuracy, safety, and relevance.

 



 
 
 * Smarter Feedback Loops: Every failed evaluation (Programmatic Rule evals) now shows why it failed, helping the refiner learn
   and your prompts improve faster.
