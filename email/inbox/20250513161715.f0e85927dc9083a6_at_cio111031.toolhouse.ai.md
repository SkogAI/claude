npm i @toolhouseai/cli -g ( https://www.npmjs.com/package/@toolhouseai/cli )
Faster Agent creation?

The CLI makes building agents incredibly easy.

CLI users define an agent in a Toolhouse agent specification file (referred to as "th files" for short). Each "th file" is a YAML file with a configuration that the CLI can read. The file contains the prompt, a CLI-generated chat ID, and default values for variables, bundles, and other metadata.

Read more ( https://docs.toolhouse.ai/toolhouse/quick-start-deploy-your-first-agent )
