We are introducing changes to our free tier.

BYOK (bring-your-own-key) is now enabled in free-tier plans, including support for OpenAI Codex (ChatGPT Plus/Pro) and zAI Coding plans (MiniMax coming soon!)

 * In Letta Code, providers can be added with /connect and available models can be viewed with /models

 * On the Letta Platform, providers can be added on the models page https://app.letta.com/projects/default-project/models

Usage credits are no longer included in free-tier plans, though select models will be available for free usage (e.g. GLM-4.7).

We have updated free-tier resource limits to 3 agents and 1 project per account.

 * You may need to delete existing agents if you are over the current limit

 * Conversations are unlimited and can be created in Letta Code with letta --new
