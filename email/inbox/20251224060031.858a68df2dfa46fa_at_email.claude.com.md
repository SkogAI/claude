Over the past two weeks, we shipped a bunch of updates to Claude Code—designed 
to help you work faster, save time, and delegate more tasks to Claude.

You can now spawn background tasks that keep running while you move on to 
other work. Browse and install plugins from a curated marketplace. Let Claude 
suggest your next prompt. And if you’re on Max, share guest passes with friends.



Here’s the full rundown: 






Work anywhere 
  

 * Claude Code on Android: Run async tasks from the Android app
 <https://claude.com/download>
 * Claude on Slack <https://claude.com/blog/claude-code-and-slack>: Delegate 
tasks and hand-off context by tagging Claude in Slack

 * Chrome integration: Validate frontend work by connecting to Claude on Chrome
 <https://claude.com/chrome> 






Smoother workflows 
  

 * Async subagents: Tasks can now spawn async subagents that move to the 
background and work independently

 * Prompt suggestions: Claude will occasionally suggest your next prompt after 
the prior one completes

 * Model switcher hotkey: Switch models mid-prompt with alt+P / option+P

 * Syntax highlighting: Diffs now support syntax highlighting for more 
languages







Personalization & insights 
  

 * Plugins marketplace: Run /plugins <https://code.claude.com/docs/en/plugins> 
for a list of ready-to-install plugins

 * Custom session names: Type /rename to give any previous session a name

 * Context info in status line: Add info on your context window usage to your 
status line
 <https://code.claude.com/docs/en/statusline>
 * Stats command: Run /stats to generate a visualization of your daily Claude 
Code usage 






One more thing 
  
Guest passes: Max users can give friends 1 week of Pro access with /passes. 



All features are live and they mean big improvements for your productivity in 
2026. Thank you again for your support this year. And as always, we look 
forward to your feedback and suggestions as we improve.

If you’re ready to get started, run `claude update` to get the latest.
