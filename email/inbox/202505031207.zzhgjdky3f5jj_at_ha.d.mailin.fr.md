Hello Dify Users,

 

We hope you’re doing well. We’re reaching out to share that Dify has officially released version 1.0.0, introducing several new features to enhance your AI application development experience. Here are the highlights:

 


###**🧩 Plugin System & Marketplace**
 

Models and tools are now decoupled from the core platform. You can install new plugins or update existing ones directly without upgrading the entire platform. You’ll also find new plugin categories—like [**Agent Strategies**](https://b6usd.r.a.d.sendibm1.com/mk/cl/f/sh/7nVU1aA2nfsTSeY62oltNKjy2JTL2Wc/MUW97JnfNSAl), [**Extensions**](https://b6usd.r.a.d.sendibm1.com/mk/cl/f/sh/7nVU1aA2nfuMSB3P1O0LPh3mCfy7mT8/CwavYoIwdCmj), and [**Bundles**](https://b6usd.r.a.d.sendibm1.com/mk/cl/f/sh/7nVU1aA2nfwFRhYhzxEnS3NaN2SuWPe/kgt4nNC-k8Y7)—to help you customize your workflows. Currently, the [**Dify Marketplace**](https://b6usd.r.a.d.sendibm1.com/mk/cl/f/sh/7nVU1aA2nfy8RE40yWTFUPhOXOxhGMA/1Tx2ncFUn8vJ) hosts more than 120 plugins covering various use cases.

 


###**🤖 Smart Workflows with Agent Nodes**


 

An [**Agent Node**](https://b6usd.r.a.d.sendibm1.com/mk/cl/f/sh/7nVU1aA2ng01QkZJx5hhWm1ChlSU0Ig/qunqQfahBDNc) is a component in Dify Workflow that enables autonomous tool invocation. By integrating different Agent reasoning strategies, LLMs can dynamically select and execute tools at runtime, allowing for multi-step reasoning in your Chatflows and Workflows.

We’ve made it easier than ever to integrate external platforms—like Slack and Discord—so you can quickly develop plugins and build AI apps that work seamlessly with other services.
