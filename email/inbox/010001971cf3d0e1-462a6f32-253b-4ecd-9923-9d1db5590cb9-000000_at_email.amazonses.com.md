PREVENTING DUPLICATE EMAILS

https://resend.com/changelog/idempotency-keys

Duplicate emails can be frustrating for your users, cost you money, and
potentially have other side effects, such as when email sending triggers other
state changes in your application. This month, we announced that the Resend
Email API now supports idempotency keys.

→ Read the changelog https://resend.com/changelog/idempotency-keys




GET CONTACT BY EMAIL API

https://resend.com/changelog/get-contact-by-email

Each contact has a unique ID, which you can use to get their contact
information. This month, we introduced an additional way to get contacts: by
their email address.

→ Read the changelog https://resend.com/changelog/get-contact-by-email




OUTLOOK & HOTMAIL CHANGES

https://resend.com/blog/microsoft-bulk-sending-requirements-2025

Starting May 5, 2025, Microsoft enforced new requirements for high-volume
senders. If you're already following best practices, this won’t be a big
adjustment. However, there are a few Microsoft-specific quirks you’ll want to
double-check. Here's a walk through of who this applies to, what’s changing, and
how to make sure you’re covered.

→ Read the post https://resend.com/blog/microsoft-bulk-sending-requirements-2025




ADVANCED DNS CONFIGURATION

https://resend.com/changelog/custom-return-path

You can now optionally set a custom return path for your domain. The custom
return path is used for SPF authentication, DMARC alignment, and handling
bounced emails.

→ Read the changelog https://resend.com/changelog/custom-return-path




WHY EMAILS GO TO SPAM

https://resend.com/blog/why-your-emails-are-going-to-spam

Emails are marked as spam when inbox providers detect signs that the message is
unwanted, suspicious, or low quality. But what can you do about it? In this
post, we share some internal tools as well as third-party tools that can help
you.

→ Read the post https://resend.com/blog/why-your-emails-are-going-to-spam




STREAMLINING DOMAIN VERIFICATION

https://resend.com/changelog/improved-dns-features

intuitive.
