Look back on LangChain's biggest launch week yet, including OSS and LangSmith updates

Last week was LangChain’s 3rd birthday – and naturally, we had to celebrate it with the most product updates we’ve ever shipped, alongside our Series B $125M funding announcement. Here’s a quick look back at what we launched.

LangSmith: Insights Agent and Multi-turn Evals

Agents can generate thousands of traces, making it hard to comb through them individually. Our new Insights Agent automatically clusters them into patterns so you can spot failure modes, prioritize high-impact issues, and build datasets for testing – all from real usage.

Once you’ve identified a pattern with Insights, use Multi-turn Evals to measure whether your agent solved the user’s problem across a full conversation, not just an individual response. Automatically evaluate task completion, user satisfaction, and agent trajectory across multi-turn threads.

LangChain 1.0 and LangGraph 1.0

Our open source frameworks are now in stable 1.0 versions, and we’re committing to no breaking changes until 2.0.

LangChain lets you build agents fast with any model provider, and is now powered by LangGraph's runtime. It includes new middleware for customization and standard content blocks. LangSmith Agent Builder (in private preview)

LangSmith Agent Builder lowers the barrier to building agents with a no-code, text-to-agent experience. Just use natural language to describe the agent you want to build (instead of a drag-and-drop workflow). The agents you create can dynamically plan and reason through tasks, with built-in tool authorization.
