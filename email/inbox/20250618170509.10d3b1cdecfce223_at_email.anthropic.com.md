Hi there,



Claude Code is now more powerful with three new features available to all 
users.



Connect to your favorite tools with remote MCP servers

Claude Code now supports remote MCP servers like Sentry and Linear so you can 
personalize and extend your coding experience without managing local MCP 
servers.Read the blog for more 
<http://anthropic.com/news/claude-code-remote-mcp>.



Plan complex work before implementing

With plan mode, you can review implementation plans for changes before Claude 
Code starts executing. Press Shift+Tab twice to activate it.



Build apps using the new Claude Code SDKs

New TypeScript and Python SDKs are now available. Use the same agent and tools 
as Claude Code to build your own agents.See the docs to get started 
<https://docs.anthropic.com/en/docs/claude-code/sdk>.



These features are available now for all Claude Code users. Run `claude update 
&& claude` to get started.
