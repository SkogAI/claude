Deploy agents as API
********************

If you know how to call an API, you now know how to build agents. The new Streaming API let you define an agent and deploy it as a streaming service.

You can integrate them in your app, add them to your vibe coded projects, or even simply run them in your browser.

Try an agent stream ( https://agents.toolhouse.ai/bdd9db4e-6dc3-4c2b-82e1-42fc71ecf67d )
We did the hard part for you

As you would expect from Toolhouse, even your deployed agents will have access to all the MCP servers, bundles, memory, code interpreter, and other tools. No boilerplates, framework, or integrations needed.

Read more ( https://docs.toolhouse.ai/toolhouse/deploy-and-run-your-agents )

**************************************
Set your agents free 🦜⛓️‍💥
**************************************

Because every agent is an API, you can parallelize or chain your agents in native code, for example by simply using fetch calls in JS/TS (or your favorite http library in Python or any other languages.) It's that simple.

Finally you can build agents, not boilerplate.
