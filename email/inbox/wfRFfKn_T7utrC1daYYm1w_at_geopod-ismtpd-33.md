T he tradeoffs come down to time and energy.

* *Time:* How long will it take you to learn the skills to pass a technical interview? How long will it take you to land a job?
* *Energy:* And, if we were all being a bit more honest, how the heck are you going to stick with all this coding stuff when you have a life?

*The bad news*

Learning backend development is not a get-rich-quick scheme. If this is something you want, you’ll have to commit to some serious hands-on keyboard time. Sorry. That’s just how it is.
