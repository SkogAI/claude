Ollama can now launch Pi [https://pi.dev/], the coding agent that powers OpenClaw.
 
Pi is designed to be a minimal coding harness that can be adapted to your workflows to create your own coding agent.
 
Pi comes bundled with powerful primitives to build on, and can be extended with extensions [https://pi.dev/packages], skills,
prompt templates, and themes.
 


Get started

Start by downloading the latest version of Ollama:
 

Download Ollama [https://ollama.com/download]

 
Next, use Ollama to launch Pi with a model (e.g. Kimi-K2.5). This installs Pi and configures Ollama as a provider including web
tools and multimodal support:
 
ollama launch pi --model kimi-k2.5:cloud
 


Customize Pi

All Pi packages [https://pi.dev/packages] work with Pi & Ollama, making it infinitely customizable for different tasks and use
cases.
 
Subagents
For tasks that involve multiple agents like deep research or large codebase refactors, use pi-subagents:
 
pi install npm:pi-subagents
 
Autoresearch
Inspired by Andrej Karpathy’s Autoresearch, the pi-autoresearch extension gives Pi the tools and workflow to run autonomous
optimization loops to solve hard problems:
 
pi install https://github.com/davebcn87/pi-autoresearch
 
Once installed, run ollama launch pi then start a loop with /autoresearch:
 
/autoresearch optimize test runtime and monitor correctness
 


Works with local and cloud models

Pi’s minimal footprint makes it a great fit for running models on your own machine with Ollama. For harder tasks, your Ollama
account includes access to larger models on Ollama’s cloud.
