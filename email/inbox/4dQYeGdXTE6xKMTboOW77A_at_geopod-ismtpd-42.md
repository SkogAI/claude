We recently released *gpt-realtime-1.5* in the *Realtime API* with stronger instruction following, more reliable tool calling, and improved multilingual accuracy for voice and real-time experiences.

We also released gpt-audio-1.5 in Chat Completions, which uses the same underlying model as the gpt-realtime-1.5 and has the same improvements.

We *recommend* updating your model selection to gpt-realtime-1.5 if you use the Responses API, or gpt-audio-1.5 for Chat Completions, to get the best performance out of our speech-to-speech models. To implement the new model, simply set the model to gpt-realtime-1.5 (for Chat Completions users, set the model to gpt-audio-1.5).
