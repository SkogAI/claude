🗄️ Agent + Your Production Database 



Now you can ask Agent questions about what your users are actually seeing.

Agent can now securely view and query your production SQL database in 
read-only mode.

That means you can:


 *  Debug issues against real user data
 * Understand live schemas instantly
 * Build features with real production context

No risk of accidental writes, just smarter decisions powered by real data.

For teams shipping production apps, this is a major win!


📬 Agent Inbox 



 Your users can now leave feedback directly on your published apps, and Agent 
can implement it.

When someone submits feedback through the built-in widget, it appears in your 
Agent Inbox 
<https://docs.replit.com/cloud-services/deployments/enable-feedback?utm_source=iterable&utm_medium=email&utm_campaign=replit-product-updates>
. From there, you can prompt Agent to make the requested improvements.

 I just added this feature to my own apps!


📌 Other new features


 * Open in Replit (Linear Beta) – Open Linear tickets directly in Replit with 
issue details automatically prefilled in your prompt
 * Custom OpenAI key support (Teams) – Use your own OpenAI API key instead of 
the default model provider 

 You can find weekly updates and more in our changelog
