Qwen3-Next-Coder is now available on Ollama with version 0.15.5. This model supports a context window of 256K tokens and was
trained specifically for agentic coding — tool use, long tasks, and recovering from errors.
 

Download model [https://ollama.com/library/qwen3-coder-next]

 
To get started, download Ollama [https://ollama.com/download] and run Qwen3-Coder-Next:
 
ollama run qwen3-coder-next
 
Note: this model requires 64GB of VRAM.
 
For those without the required hardware, this model is also available to run on Ollama's cloud using your ollama.com account:
 
ollama signin
ollama run qwen3-coder-next:cloud
