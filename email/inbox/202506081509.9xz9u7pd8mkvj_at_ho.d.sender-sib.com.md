New Features:

###[**Secure Access with NetBird + Microsoft Intune**](https://az5n7.r.ag.d.sendibm3.com/mk/cl/f/sh/1t6Af4OiGsEag0xCMqL2XlS3Ztc5EU/8BRnvZLE8j-W)


NetBird now integrates with Microsoft Intune, allowing you to enforce network access only for compliant Windows and macOS devices. This ensures that only up-to-date and secure endpoints can connect to your protected resources—based on compliance policies like OS version, device health, and security baseline adherence.

###[**New Onboarding Flow: Zero to Mesh in Minutes**](https://az5n7.r.ag.d.sendibm3.com/mk/cl/f/sh/1t6Af4OiGsGQ0rktwGijCnhN7BX24k/TcJcxIEMEVAW)


Getting started with NetBird has never been faster. Follow our updated onboarding guide to install the agent, create your network, and connect peers with ease.


###[**Switching Between Profiles Just Got Easier**](https://az5n7.r.ag.d.sendibm3.com/mk/cl/f/sh/1t6Af4OiGsHKgI9kiyuZXJp1spUVUs/vdHysaI1V9UG)

###[**New: DNS0 Name Server Template Support** - (PR #144)](https://az5n7.r.ag.d.sendibm3.com/mk/cl/f/sh/7nVU1aA2nfsZpYhz5TNontOL9tnQTm2/OGlyEBbYWVkl)

###[**Release v2.37.0: Better Control for Frontend/Backend Traffic**](https://az5n7.r.ag.d.sendibm3.com/mk/cl/f/sh/7nVU1aA2ng3tmhjowsmX23JCA4m8vR8/1zWsO_L9hj0f)
