Debugging Network Errors Just Got 10x Better
--------------------------------------------

Connecting to APIs and external data sources can be tricky. We’ve made it easier.
Lovable now reads network logs directly, using them as context to debug and implement third-party integrations more effectively.
No more disruptive error pop-ups—Lovable now uses real-time network insights to help you fix issues faster and keep building.

-----------------------------------------
Chat Mode Improvements and pricing update
-----------------------------------------

We launched Chat Mode as a Labs
feature some weeks ago, and the feedback from early testers has been overwhelmingly positive. With recent upgrades, Chat Mode now feels like talking to a seasoned CTO—one who knows your projects and goals inside out.
However, these advanced capabilities come with additional costs. Starting now, messages in Chat Mode will count toward your message limit. While we initially announced this update in our Discord, we recognize that a broader announcement was needed to ensure everyone stayed informed.

We’re actively exploring ways to optimize these costs and make Chat Mode more affordable.
If you haven’t tried it yet, you can enable Chat Mode today by heading to Account > Settings > Labs.
