What’s new in AI
----------------

* GitHub Copilot coding agent (Preview) ( https://github.blog/changelog/2025-05-19-github-copilot-coding-agent-in-public-preview/?utm_source=github&utm_medium=email&utm_campaign=mpu_june2025 ) : Your backlog doesn’t stand a chance! Assign issues to Copilot, and it will work in the background to automatically implement features, fix bugs, and open pull requests for your review.
* Now is the time to migrate to GitHub ( https://resources.github.com/accelerate-innovation-azure-devops-migration/?utm_source=github&utm_medium=email&utm_campaign=mpu_june2025 ) : Harness the power of agentic AI by migrating your Azure DevOps repositories to GitHub while preserving your existing Azure Boards and Pipelines workflows.
* Agent mode with MCP expands to JetBrains, Eclipse, and Xcode ( https://github.blog/changelog/2025-05-19-agent-mode-and-mcp-support-for-copilot-in-jetbrains-eclipse-and-xcode-now-in-public-preview/?utm_source=github&utm_medium=email&utm_campaign=mpu_june2025 ) : All three of these IDEs now feature Copilot agent mode, enabling you to run multi-step processes, analyze your codebase, run terminal commands, and detect and fix errors.
* GitHub Models in your repositories ( https://github.blog/changelog/2025-05-19-github-models-built-into-your-repository-is-in-public-preview/?utm_source=github&utm_medium=email&utm_campaign=mpu_june2025 ) : We’ve added prompt management, lightweight evaluations and enterprise controls to GitHub Models so teams can experiment with best-in-class models, without leaving GitHub.

( https://github.blog/changelog/2025-05-29-introducing-copilot-spaces-a-new-way-to-work-with-code-and-context/?utm_source=github&utm_medium=email&utm_campaign=mpu_june2025 )

GitHub Copilot Spaces Public Preview
------------------------------------

Copilot Spaces ( https://github.blog/changelog/2025-05-29-introducing-copilot-spaces-a-new-way-to-work-with-code-and-context/?utm_source=github&utm_medium=email&utm_campaign=mpu_june2025 ) is now available to anyone who wants to:

* *Make Copilot into a subject matter expert* : Copilot Spaces enables you to group related code, documentation, issues, and other contexts to produce focused, high-quality AI-powered answers and suggestions.
* *Scale expertise across teams* : Share expertise by creating spaces that teammates can access, giving them instant access to specialized knowledge and saving valuable development time.
* *Always up to date* : Files and repositories added to a space are always kept up to date as they change, making Copilot an evergreen expert in your project.

( https://github.blog/changelog/2025-05-12-github-enterprise-cloud-data-residency-in-the-us-is-generally-available/?utm_source=github&utm_medium=email&utm_campaign=mpu_june2025 )

Data Residency in the US is generally available
-----------------------------------------------

GitHub Enterprise Cloud Data Residency ( https://github.blog/changelog/2025-05-12-github-enterprise-cloud-data-residency-in-the-us-is-generally-available/?utm_source=github&utm_medium=email&utm_campaign=mpu_june2025 ) provides:

* *Flexibility to choose where your code and repository data is stored* , while maintaining full GitHub capabilities.
* *Secure, Azure-powered infrastructure* for enhanced security for your code in transit and at rest.
* Data residency is also available in the European Union and Australia ( https://github.blog/changelog/2025-05-16-github-enterprise-cloud-with-data-residency-now-available-for-self-service-trial/?utm_source=github&utm_medium=email&utm_campaign=mpu_june2025 ) for a self-service trial.
