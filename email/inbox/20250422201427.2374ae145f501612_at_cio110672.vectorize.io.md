Welcome to the latest Vectorize newsletter! 🎉 We're sharing new features, AI engineering insights, and useful resources to help you build smarter, more efficient AI applications.

 ✨ What's New

 

Chat Completions Endpoint (Beta)
All pipelines now include an OpenAI-compatible chat completions endpoint — no adapter required. Connect your chatbot, agent, or frontend UI and serve responses directly from your private data. Perfect for agents, chatbots, or any custom UI.

🧪 Read the docs ( https://docs.vectorize.io/rag-pipelines/openai-compatible-endpoint?utm_source=vectorize-newsletter&utm_medium=email&utm_campaign=newsletter-2025-04-22&utm_content=whats-new )

 

Chunk Inspection

You can now inspect the exact chunks your pipeline is retrieving directly in the Vectorize UI. See exactly how your documents are being processed — before you deploy. Great for tuning extraction and chunking strategies.

🔍 Learn how ( https://docs.vectorize.io/rag-pipelines/management#viewing-chunks?utm_source=vectorize-newsletter&utm_medium=email&utm_campaign=newsletter-2025-04-22&utm_content=whats-new )

 

Google Drive OAuth
No more service accounts. With OAuth, you can authenticate using your Google account, pick files through a built-in file picker, and deploy your pipeline in just a few clicks.

🚀Learn more ( https://vectorize.io/google-drive-oauth-a-simpler-way-to-bring-your-docs-into-vectorize?utm_source=vectorize-newsletter&utm_medium=email&utm_campaign=newsletter-2025-04-22&utm_content=whats-new )

 

Set Spending Limits on Serverless

Control your costs with spending limits for the serverless platform. You can set soft or hard caps to avoid surprise overages.
💸Learn more ( https://docs.vectorize.io/administration/billing-and-usage/spending-limits?utm_source=vectorize-newsletter&utm_medium=email&utm_campaign=newsletter-2025-04-22&utm_content=whats-new )

 

GitHub Connector
Bring in GitHub issues and pull requests as part of your RAG pipeline. Filter by labels and status to index the metadata that matters.
🛠 Read the docs ( https://docs.vectorize.io/integrations/source-connectors/github?utm_source=vectorize-newsletter&utm_medium=email&utm_campaign=newsletter-2025-04-22&utm_content=whats-new )

 

Notion Connector

Pull in content from Notion pages and databases using OAuth — no exports, no copy-pasting. Just select what you need and connect it to your pipeline.

📘 Learn more ( https://docs.vectorize.io/integrations/source-connectors/notion?utm_source=vectorize-newsletter&utm_medium=email&utm_campaign=newsletter-2025-04-22&utm_content=whats-new )
