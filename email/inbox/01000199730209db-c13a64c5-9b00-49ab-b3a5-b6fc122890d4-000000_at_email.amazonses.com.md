September 2025 Updates


📓 Modal Notebooks



Modal Notebooks allow you to experiment and collaborate on Python code in Modal’s cloud, no setup required. Attach custom images
and distributed volumes, accelerate your code with up to 8 B200 GPUs, collaborate in real time, and share your work on the web.
 
To get started, open modal.com/notebooks [https://modal.com/notebooks].
 

Read more [https://modal.com/blog/notebooks]

 


⌛ Sandbox Idle Timeouts

You asked, we listened. Users are now able to provide an optional idle_timeout parameter to modal.Sandbox.create(). When provided,
Sandboxes will terminate after a specified number of seconds of idleness.
 

More details [https://modal.com/docs/guide/sandbox#idle-timeouts]

 


👩‍💻 Client updates

Run pip install --upgrade modal to get the latest client updates. Here are some highlights from the changelog
[https://modal.com/docs/reference/changelog]:
 * Added a startup_timeout parameter to the @app.function() and @app.cls() decorators. When used, this configures the timeout
   applied to each container’s startup period separately from the input timeout (1.1.4
   [https://modal.com/docs/reference/changelog#114-2025-09-03])



 * Introduced a new API pattern for imperative management of Modal resource types (modal.Volume, modal.Secret, modal.Dict, and
   modal.Queue) (1.1.2 [https://modal.com/docs/reference/changelog#112-2025-08-14])

🏎️ GPU Performance Glossary



For GPUs and their applications, performance is the product. That's why we extended our popular "CUDA docs for humans", the GPU
Glossary, with a new section on performance [https://modal.com/gpu-glossary/perf]. We cover everything from warp divergence at the
assembler level [https://modal.com/gpu-glossary/perf/warp-divergence] to memory-bound LLM inference napkin math
[https://modal.com/gpu-glossary/perf/memory-bound] to the roofline model from quantitative computer architecture
[https://modal.com/gpu-glossary/perf/roofline-model].
 

Read more [https://modal.com/gpu-glossary/perf]

 


🦥 Finetune an LLM with Unsloth

Unsloth [https://unsloth.ai/] provides optimized methods for LLM finetuning with LoRA and quantization, leading to 2x faster
training with 70% less memory usage. Learn how to use Unsloth to finetune a version of Qwen3-14B with the FineTome-100k dataset on
Modal using only a single GPU!
 

Check out the example [https://modal.com/docs/examples/unsloth_finetune]



🗣️ Finetune an ASR model with Whisper

In this example, we demonstrate how to fine-tune OpenAI’s Whisper [https://huggingface.co/openai/whisper-tiny.en] model to improve
transcription accuracy.
 

Learn more [https://modal.com/docs/examples/fine_tune_asr]
