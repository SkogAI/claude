Enroll in LangChain Academy and learn how to build an ambient agent with LangGraph

We’re excited to launch our latest LangChain Academy course: Building Ambient Agents with LangGraph.

Most agents today operate through chat interfaces, handling one request at a time. But as models have improved, agents can now run in the background – triggered by events like Slack messages or GitHub issues – and take on long-running, complex tasks. These “ambient agents” still benefit from human oversight to review actions, answer questions, and guide improvements.

We designed LangGraph to support long-running, stateful agents with built-in support for memory and human-in-the-loop workflows. LangGraph Platform provides the infrastructure to run these agents at scale, while LangSmith offers the tools to observe, evaluate, and improve them. Together, they make it easier to build reliable, production-ready agents for complex tasks.

In this course, you'll:

- Learn the fundamentals of LangGraph and how to build your own email agent
- Evaluate your agent using LangSmith
- Add human-in-the-loop for reviews, and memory so your agent adapts over time
- Deploy your agent with LangGraph Platform and connect it to Gmail
