INTEGRATIONS

OpenAI Agent SDK (TypeScript)

With the OpenAI Agents SDK (TS), building agents in TypeScript just got easier - no Python, no context switching, just native tooling and agent workflows.

The SDK is great for building; but in production, it lacks logging, retries, cost controls, and provider flexibility.
Portkey fills those gaps with observability, guardrails, and multi-provider support, built in. Learn more -> ( https://portkey.ai/docs/integrations/agents/openai-agents-ts?cr_email=ZW1pbC5za29nc3VuZEBnbWFpbC5jb20%3D&utm_campaign=Jun%2725+Updates&utm_content=Jun+25+Updates&utm_medium=email_action&utm_source=customer.io )

Cursor

Bring visibility and governance to Cursor’s coding assistant with Portkey. Choose from 1600+ models, track requests, enforce rate limits, and log usage, making it easy for enterprises to govern Cursor across the org.
Learn more -> ( https://portkey.ai/docs/integrations/libraries/cursor?cr_email=ZW1pbC5za29nc3VuZEBnbWFpbC5jb20%3D&utm_campaign=Jun%2725+Updates&utm_content=Jun+25+Updates&utm_medium=email_action&utm_source=customer.io#cursor )

Strands Agents

Enterprises building on Strands Agents now have a cleaner path to production — thanks to Portkey as the LLM gateway. With a single abstraction layer, you can:

* Access 2,000+ LLMs with provider-agnostic logic
* Standardize tool calling across providers
* Add conditional routing for cost control
* Enable retries, rate limits, and full observability

Strands Agents PR ( https://github.com/strands-agents/sdk-python/pull/197?cr_email=ZW1pbC5za29nc3VuZEBnbWFpbC5jb20%3D&utm_campaign=Jun%2725+Updates&utm_content=Jun+25+Updates&utm_medium=email_action&utm_source=customer.io )
