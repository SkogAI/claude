Hey there, here's what's new in Lovable:

Claude Opus 4.6 now in Lovable ( https://lovable.dev/blog/opus-4-6-now-in-lovable )

Now powered by Claude Opus 4.6

Claude Opus 4.6 is now a core model in Lovable. It scores 21% higher on our app building benchmark, runs twice as long on complex tasks, and is noticeably stronger on design-heavy work like landing pages and marketing sites. Read more ( https://lovable.dev/blog/opus-4-6-now-in-lovable )

A smarter agent

The agent now thinks before it builds (Plan Mode), handles multiple tasks in sequence (queuing), and can test your app like a real user (browser testing). See what changed ( https://lovable.dev/blog/a-smarter-lovable )

Test & live environments

Now you can build and experiment in Test without touching your production app. When you're ready, publish to Live. Your test data stays separate, so you can break things without breaking things for real users. Free during Beta. Docs ( https://docs.lovable.dev/features/environments )

Google sign-in

Add Google sign-in with a single prompt. Lovable builds the full auth flow so your users get one-tap login and you don't have to deal with passwords. Docs ( https://docs.lovable.dev/features/google-auth )
