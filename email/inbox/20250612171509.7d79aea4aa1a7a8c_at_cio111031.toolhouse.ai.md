Get started
-----------

You know we are all about simplicity. Getting started is super easy!

*
Grab an API key from your provider. Groq is our speed champion, but OpenAI and Anthropic work great too
* Store it securely: th secrets set GROQ_API_KEY=your_key
* Add the model to your th file
* Test with th run and deploy with th deploy
*
That's it! You're now running with your model of choice. 🚀
